# 統計画面：耐性スコア分布（Resistance Score Distribution）仕様

最終更新日時: 2026-02-02

## 1. 概要
統計画面（Statistics）に表示される「耐性スコア分布」グラフは、ユーザーが学習中のカードが「どれだけ長期的な記憶として定着しつつあるか」を可視化したものです。

以前の「記憶の定着度（Retention Probability）」は復習予定日までの時間経過で減少する値でしたが、この「耐性スコア（Resistance Score）」は**復習間隔（Interval）が伸びるほど積み上がっていくスコア**であり、学習の蓄積実感を得やすい指標となっています。

## 2. 表示要素

### 2.1 分布グラフ（Bar Chart）
- **X軸（横軸）**: 耐性スコア（0% 〜 100%）
  - 左側（0%）: 新規カード、復習間隔が短いカード
  - 右側（100%）: 長期記憶、復習間隔が長いカード（最大90日以上）
- **Y軸（縦軸）**: カード枚数
- **色分け（フェーズ）**:
  スコアに応じてバーの色が変化します（グラデーション表示）。
  - **0-20% (赤)**: 要復習 / 新規
  - **20-40% (オレンジ)**: 覚えかけ
  - **40-65% (黄)**: 定着途上
  - **65-85% (水色)**: 安定
  - **85-100% (緑)**: 長期保持

### 2.2 平均耐性スコア
- グラフ上に点線で表示される垂直ライン。
- 計算式: 全カード（復習済みのみ）の耐性スコアの平均値。
- 表示形式: `平均耐性スコア: XX%`

## 3. 計算ロジック

耐性スコア（$R$）は、各カードの現在の復習間隔（$I$ = 日数）に基づき、対数スケールで算出されます。

### 3.1 計算式
$$
R = \text{round} \left( \frac{\log_{10}(1 + I)}{\log_{10}(1 + \text{MAX\_INTERVAL})} \times 100 \right)
$$

- $I$ (Interval): 現在設定されている次回までの復習間隔（日数）。復習直後であれば、次の復習までの日数。
- $\text{MAX\_INTERVAL}$: 最大スコア（100%）に達する基準日数。現在は **90日** に設定されています。

### 3.2 スコアと日数の対応例
| 復習間隔 ($I$) | 耐性スコア ($R$) | 意味合い |
| :--- | :--- | :--- |
| 0日 | 0% | 未定着・即時復習 |
| 1日 | 約 15% | 初回学習直後 |
| 3日 | 約 30% | 短期記憶 |
| 7日 | 約 46% | 1週間保持 |
| 30日 | 約 76% | 1ヶ月保持 |
| 90日以上 | 100% | 3ヶ月以上（長期記憶完了） |

## 4. データソース
- **対象データ**: 復習履歴のあるカード（`isReviewed(card) == true`）のみが集計対象です。未学習カードは含まれません。
- **更新タイミング**: カードの復習を行い、復習間隔（Interval）が更新されると即座にスコアも再計算され、グラフ上の位置が右側へ移動します。
