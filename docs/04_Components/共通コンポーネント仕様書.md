# 共通コンポーネント (Shared Components) リファレンス

## 概要
アプリケーション全体で使用される、再利用可能なUIコンポーネントおよびユーティリティコンポーネントの仕様です。
主に `src/Components/ui` および `src/Components/common` に配置されています。

## UI プリミティブ (Base UI)
`Radix UI` をベースに `Tailwind CSS` でスタイリングされた基本コンポーネント群です。

- **Button**: バリアント (`default`, `outline`, `ghost`, `destructive`) を持つボタン。
- **Dialog / Modal**:
    - `SettingsDialog`: 設定画面用。
    - `FolderDialog`: フォルダ作成・編集用。
    - `DeleteConfirmDialog`: 削除確認モーダル。
- **Toast**: ユーザーへのフィードバック通知（成功、エラー、情報）。`ToastContext` 経由で呼び出し。

---

## 🎨 デザインシステム (Design System & Tokens)

アプリケーション全体で一貫したブランド体験を提供するための規約です。

### 1. アクセントカラー (Dynamic Primary)
ユーザー設定により動的に変更されます。
- **Tailwind Class**: `primary-50` ~ `primary-900` を使用。
- **CSS変数**: `--color-primary-600-hex` 等に現在の色がセットされます。
- **適用先**: ボタン、アイコン、アクティブなタブ、プログレスバー等。

### 2. インタラクション
- **Hover**: 基調色への変化や `scale-105` などの微細なアニメーション。
- **Glassmorphism**: ヘッダーやポップアップには `bg-white/90` と `backdrop-blur-md` を併用。

## カード表示系コンポーネント

### 1. `Flashcard`
アプリケーション全体でカードを描画するための統一コンポーネントです。
- **場所**: `src/Components/card/Flashcard.tsx`
- **使用箇所**: `CardViewer` (一覧閲覧), `StudyCard` (学習), `CardEditor` (プレビュー)。
- **機能**:
    - 問題/解答の切り替えアニメーション。
    - 数式・コードブロックのレンダリング。
    - メディア（画像・音声）の統合表示。
    - ブックマーク、曖昧マーク、タグの表示。

### 2. `CardMedia`
カードに付随するメディア表示ロジックをカプセル化したコンポーネント群です。
- **場所**: `src/Components/card/CardMedia.tsx`
- **内容**:
    - `AudioPlayer`: 複数音源の再生管理。
    - `ImageGallery`: 画像のグリッド表示とフルスクリーン表示。

## 機能コンポーネント

### 1. `MediaUploader`
画像や音声ファイルのアップロードを処理するドラッグ＆ドロップ対応エリア。
- **Props**:
    - `onFilesSelected`: ファイル選択時のコールバック。
    - `existingFiles`: 初期表示するファイルリスト。
    - `maxFiles`: 最大ファイル数制限。
- **機能**:
    - プレビュー表示。
    - クリップボードからのペーストイベント検知。
    - 削除ボタン。

### 2. `SyncStatusIndicator`
サイドバーなどに配置され、現在のクラウド同期状態をアイコンで示します。
- **状態**:
    - 🟢 (緑点灯): 同期完了。
    - 🔵 (青回転): 同期中。
    - 🔴 (赤): エラー。
    - ⚪ (グレー): オフライン。
- クリックすると詳細なステータスダイアログまたは手動同期をトリガー。

### 3. `DraggableCard` (`@hello-pangea/dnd`)
カレンダーや並べ替えリストで使用される、ドラッグ可能なカードアイテム。
- 長押しでドラッグ開始。
- ドロップエリア (`Droppable`) とセットで使用。

### 4. `HandTray` (マップ用カードトレイ)
- **場所**: `src/Components/map/HandTray.tsx`
- **機能**: マップ未配置のカードを水平スクロールで表示。ドラッグしてマップに投入。

### 5. `CardPopup` (マップ詳細)
- **場所**: `src/Components/card/CardPopup.tsx`
- **機能**: マップノードをタップした際に表示されるモーダル。`Flashcard` をラップし、裏表反転をサポート。

### 4. `ErrorBoundary`
Reactコンポーネントツリー内のエラーをキャッチし、クラッシュ画面の代わりにフォールバックUI（「問題が発生しました」など）を表示します。
- 開発モードでは詳細なスタックトレースを表示。
- 本番では「リロード」ボタンを提供。
