# ショートカットキー説明の設定画面への追加

設定画面に新しく「ショートカット」タブを追加し、アプリ内で利用可能なキーボードショートカットの一覧を表示します。

## 変更内容

### 1. 設定画面 (`src/Components/settings/SettingsDialog.jsx`)
- `sidebarItems` に「ショートカット」を追加し、`Keyboard` アイコン（lucide-react）を割り当てます。
- `renderContent` 内に `shortcut` タブの表示ロジックを追加します。
- ショートカットキーをカテゴリー別（全般、学習、カレンダー、エディタ）に分類して、見やすくレイアウトされたテーブルやリストで表示します。

### 2. インポートの追加
- `lucide-react` から `Keyboard` アイコンをインポートします。

## 表示するショートカット一覧

### 全般 (Global)
- `H`: ホームに移動
- `C`: カレンダーに移動
- `S`: 統計に移動
- `,` (カンマ): 設定を開く
- `T`: 訓練モードに移動
- `?`: ヘルプを表示
- `Ctrl + B`: サイドバーの開閉

### 学習モード (Study Mode)
- `1` / `O`: 覚えた
- `2` / `X`: 忘れた
- `3` / `S`: スキップ
- `Space` / `Enter`: 解答を表示

### カレンダー (Calendar)
- `←` / `→`: 日付の移動
- `↑` / `↓`: 週の移動

### カードエディタ (Card Editor)
- `Tab`: 入力項目の移動 (タイトル → 問題 → 解答)
- `Shift + Tab`: 前の入力項目へ移動

## 検証計画

### 自動テスト
- 現時点では表示のみのため、手動確認を優先します。

### 手動確認
1. 設定画面を開き、「ショートカット」タブが表示されることを確認。
2. タブをクリックして、全てのショートカットキーが正しく説明されていることを確認。
3. ダークモード/ライトモード、モバイル/デスクトップ表示での崩れがないか確認。
