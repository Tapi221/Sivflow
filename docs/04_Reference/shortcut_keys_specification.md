# ショートカットキー仕様書

本ドキュメントでは、Flashcard Masterで利用可能なキーボードショートカットキーについて記載します。

## 1. 全般 (Global)
アプリのどの画面からでも利用可能なショートカットです(入力フィールドにフォーカスがある場合を除く)。

| キー | アクション |
| :--- | :--- |
| `H` | ホーム画面に移動 |
| `C` | カレンダー画面に移動 |
| `S` | 統計画面に移動 |
| `,` (カンマ) | 設定画面を開く |
| `T` | 訓練モードに移動 |
| `?` (`Shift + /`) | ヘルプを表示 (将来実装予定) |
| `Ctrl/Cmd + B` | サイドバーの開閉 (全般 / 作業ビュー) |
| `Ctrl/Cmd + P` | Quick Open - カード/フォルダ/タグを検索 |
| `Ctrl/Cmd + Shift + F` | Global Search - 全文検索 |

## 2. 学習モード (Study Mode)
カードの学習中に利用可能なショートカットです。

| キー | アクション |
| :--- | :--- |
| `Space` / `Enter` | 解答を表示 |
| `1` / `O` | 「覚えた」として記録 |
| `2` / `X` | 「忘れた」として記録 |
| `3` / `S` | スキップ |

## 3. カレンダー (Calendar)
カレンダー画面で日付を選択する際に利用可能なショートカットです。

| キー | アクション |
| :--- | :--- |
| `←` / `→` | 前の日 / 次の日を選択 |
| `↑` / `↓` | 前の週 / 次の週を選択 |

## 4. カードエディタ (Card Editor)
カードの作成・編集画面で利用可能なショートカットです。

| キー | アクション |
| :--- | :--- |
| `Tab` | 次の入力項目へ移動 (タイトル → 問題 → 解答) |
| `Shift + Tab` | 前の入力項目へ移動 |

## 5. フォルダカラムビュー (Folder Column View)
フォルダカラムビューで利用可能なショートカットです。

### ナビゲーション系
| キー | アクション |
| :--- | :--- |
| `↑` | 前のフォルダを選択 |
| `↓` | 次のフォルダを選択 |
| `→` | フォルダを開く |
| `←` | 親フォルダへ戻る |
| `Alt + ←` | 履歴: 前の場所 |
| `Alt + →` | 履歴: 次の場所 |

### フォルダ・カード操作系
| キー | アクション |
| :--- | :--- |
| `Ctrl/Cmd + N` | 新規フォルダ |
| `Ctrl/Cmd + Shift + N` | 新規カード |
| `Enter` / `F2` | 名前変更 |
| `Delete` / `Backspace` | 削除 |

## 備考
- 入力フィールド (Input, Textarea) にフォーカスがある間は、全般ショートカットは無効化されます。
- エディタ内での `Tab` キー移動は、主要な入力フィールド間をスムーズに移動するためのカスタム挙動です。
- フォルダカラムビューのショートカットキーは、フォルダ画面でカラムビューが表示されている場合のみ有効です。
