# XLSX一括インポート機能の拡張計画書（既存セットへの追加対応）

## 目的

現在、新規カードセットの作成のみに対応しているXLSXインポート機能を拡張し、既存のカードセットへのカード追加を選択できるようにします。また、インポート時のカード属性（順序、レイアウト等）を適切に設定します。

## 変更内容

### 1. `src/features/import/importCardsFromPayload.ts`

- `ImportDestination` 型を定義し、新規作成（`new-card-set`）または既存追加（`existing-card-set`）を受け取れるように拡張。
- インポート処理内で、宛先に応じたカードセットIDを解決するロジックを実装。
- カード生成時に以下のフィールドを明示的に設定：
  - `orderIndex`: 取り込み順に連番を設定（既存カードの後ろに並ぶよう配慮）。
  - `layoutRows`: 標準のレイアウト（4x4）を設定。
  - プロパティの初期値（`frontRows`, `backRows`, `tags` 等）を補完。

### 2. `src/features/import/ui/XlsxImportDialog.tsx`

- インポートダイアログに「取り込み先」の選択UI（`Select`コンポーネント）を追加。
- フォルダ内の既存カードセット一覧を表示し、選択可能にする。
- カードセットが存在しない場合は自動的に「新規作成」モードを固定。
- インポート実行時に、選択されたモードとカードセット情報を `importCardsFromPayload` に渡すよう修正。

### 3. `src/components/folder/layout/TreeViewLayout.tsx`

- 現在表示中のフォルダに属するカードセット一覧を `useMemo` で抽出し、`XlsxImportDialog` にプロパティとして渡すよう修正。

## 影響範囲

- XLSXインポート機能全般
- カードセット管理
- フォルダエクスプローラー

## 確認事項

- [ ] 既存のカードセットを選択して正常に追加インポートできること
- [ ] 追加インポート時、既存のカードの後に新しいカードが並ぶこと
- [ ] インポートしたカードのレイアウトが正しく設定されていること
