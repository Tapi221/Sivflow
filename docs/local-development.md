# Local development

## Install

- Node.js `>= 20.19.0` (see `package.json#engines`)
- `npm ci`

## Environment variables

See `docs/environment-variables.md`.

Quick start:

1. Copy `.env.example` to `.env.local`
2. Fill in the placeholders

## Run (web)

- `npm run dev`

## Run (desktop / Electron)

- `npm run dev:electron`
