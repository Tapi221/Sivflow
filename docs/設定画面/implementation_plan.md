# ログアウトボタン重複削除 実装計画

## 目標
設定画面 (`SettingsDialog`) 内における「アカウント設定」タブと「サイドバー」の両方に存在するログアウトボタンのうち、アカウント設定タブ内のものを削除し、UIを整理する。

## ユーザーレビューが必要な事項
特になし。

## 検証計画

### 手動検証
1. **設定画面の表示**
   - アプリを起動し、設定画面を開く。
2. **アカウントタブの確認**
   - 「アカウント」タブ（デフォルト表示）を確認し、最下部にあった「ログアウト」ボタンが削除されていることを確認する。
3. **サイドバーの確認**
   - 設定画面左側のサイドバー最下部に「ログアウト」ボタンが存在し、引き続き利用可能であることを確認する。
4. **機能確認**
   - サイドバーの「ログアウト」ボタンをクリックし、正常にログアウトできることを確認する。
