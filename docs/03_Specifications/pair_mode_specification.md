# ペアモード 基本仕様書 (Rev.2)

ペアモードは、1枚のカード内で複数の「用語（問題）」と「対応する意味（解答）」をセットで管理するための学習モードです。

## 1. レイアウト構成

基本的な配置は「一問一答モード」や他の標準モードと同一の構造を維持し、デスクトップ環境での広い画面を有効活用します。

- **エディタ画面の基本構造**: 
  - **デスクトップ（lg以上）**: 左側に「問題セクション」、右側に「解答セクション」を配置する **2カラム・サイドバイサイド形式**。
  - **モバイル**: 画面幅に応じて、垂直スタック（縦並び）に自動的に切り替わるレスポンシブデザイン。
- **デザインの一貫性**: 各ブロックのパディング（`p-2`）、間隔（`space-y-3`）、カードの角丸などは標準モードと完全に一致させます。

## 2. ブロック管理と整合性

ペアモードでは、左側の $N$ 番目のブロックと、右側の $N$ 番目のブロックが常に一対一で対応します。

- **ペア同期**:
  - **追加**: 「ペアを追加」ボタンをクリックすると、問題側と解答側にそれぞれ1つずつテキストブロックが追加されます。
  - **削除**: 左右どちらかのペアを削除すると、対になる反対側のブロックも自動的に削除されます。
- **初期状態**: 新規作成時は、初期状態で2つのペア（左2：右2）が配置された状態で開始します。

## 3. ユーザーインターフェース (UI)

### 動的プレースホルダー
各ブロックがどのペアに属するかを視覚的に明示するため、インデックスに基づいたプレースホルダーを表示します。

- **問題（左側）**: `ペア [N]：用語・単語`
- **解答（右側）**: `ペア [N]：意味・説明`

### 追加ボタン
- **配置**: 全てのブロック（解答側のセクション末尾）の下部に配置。
- **ラベル**: `＋ ペアを追加`
- **デザイン**: 周囲のデザインに馴染む `outline` スタイルのボタン。

## 4. 学習時の挙動

- **カードのめくり**: タップで反転します。
- **表示内容**: 
  - オモテ面：全ペアの問題ブロックを一覧表示。
  - ウラ面：全ペアの解答ブロックを一覧表示。
