# 画面遷移図

Flashcard Master の詳細な画面遷移構造とルーティング定義を記述します。

## 概要

アプリケーションは `Layout` コンポーネントを基盤としたシングルページアプリケーション（SPA）です。
`react-router-dom` によるクライアントサイド・ルーティングと、各種の状態管理（Context, Local State）によって画面遷移が制御されます。

## 状態・モードの詳細説明

### 1. 学習モード (Study Flow)

学習画面 (`StudyMode`) 内部の状態遷移と、完了後の分岐です。

**URLパラメータによるモード分岐:**

- `?folderId={id}`: 指定フォルダのみを学習対象とする。
- 指定なし: 「今日の復習」として、期日が到来した全カードを学習。

---

### 2. コンテンツ管理フロー (Folder & Card)

フォルダとカードの閲覧 (`FolderView`)、作成、編集、整理に関するフローです。

**FolderView 内のタブ構造:**

- **Cards Tab**: カード一覧と操作
- **Memo Tab**: フォルダ単位の自由なメモ
- **Notes Tab**: PDF資料等の閲覧
- **Map Tab**: 関連情報の視覚的マッピング

---

### 3. 設定・データ管理フロー

`SettingsDialog` 内部のタブ切り替えと、そこから派生するデータ操作です。

- **Account**: アカウント情報、ユーザー名変更
- **Study**: 学習アルゴリズム、言語設定
- **Theme**: デザインテーマ（アクセントカラー等）
- **Tags**: タグのマスタ管理
- **Sync**: クラウド同期の状態確認
- **Data**: エクスポート/インポート、データ復元、正規化

## 主要ダイアログ・モーダル一覧

| コンポーネント名 | 呼び出し元 | 説明 | 主要アクション |
| :--- | :--- | :--- | :--- |
| `SettingsDialog` | `Layout` (Header) | アプリ全体の設定、アカウント情報 | ユーザー名変更、テーマ切替、サインアウト |
| `FolderDialog` | `Folders`, `FolderView` | フォルダの作成・編集 | 作成、名称変更、色変更、削除 |
| `TagManagerDialog` | `SettingsDialog`, `Folders` | タグのマスタ管理 | タグ作成、色変更、削除 |
| `ExportDialog` | `Dashboard` | 学習データのエクスポート | JSONファイルのダウンロード |
| `ImportDialog` | `Dashboard` | 学習データのインポート | JSONファイルの読み込み、DB反映 |
| `SyncErrorDialog` | `SyncStatusIndicator` | 同期エラーの詳細表示 | エラーログ確認、再試行 |
| `LanguageSelect` | `Login`, `UncertainMode` | 学習対象言語の選択 | 言語設定の保存 |

## URLルーティング詳細

`src/App.tsx` で定義されている全ルートと、その役割の詳細です。

| パス | コンポーネント | 認証 | 詳細説明 |
| :--- | :--- | :--- | :--- |
| `/` | `Dashboard` | 要 | ルートアクセス時、ログイン済みならDashboardを表示。 |
| `/login` | `LoginPage` | 不要 | 未ログイン時はここへリダイレクトされる。 |
| `/Dashboard` | `Dashboard` | 要 | **ホーム画面**。学習サマリー、ストリーク、本日のタスク。 |
| `/folders` | `Folders` | 要 | 全フォルダのリスト表示。階層構造のルート。 |
| `/gallery` | `Gallery` | 要 | ギャラリー画面（画像一覧など）。 |
| `/world-map` | `WorldMap` | 要 | **世界地図**。PC版サイドバーからアクセス。モバイル版はナビゲーション非表示。 |
| `/calendar` | `Calendar` | 要 | 学習履歴と予定のカレンダー表示。 |
| `/statistics` | `Statistics` | 要 | 学習効率、定着率などのグラフ分析。 |
| `/trash` | `Trash` | 要 | 削除されたアイテムの保管庫。復元が可能。 |
| `/FolderView` | `FolderView` | 要 | **フォルダ詳細**。クエリ `?id={folderId}` が必須。カード一覧、メモ、資料。 |
| `/CardEdit` | `CardEdit` | 要 | **カード編集**。`?id={cardId}` で編集、なしで新規作成。`?folderId` で親指定。 |
| `/CardView` | `CardView` | 要 | **カードプレビュー**。編集せず閲覧のみ行うモード。 |
| `/study` | `StudyMode` | 要 | **学習実行**。`?folderId` があればそのフォルダ、なければ「今日の復習」。 |
| `/uncertain` | `UncertainMode` | 要 | **弱点克服**。定着度が低いカードを重点的に学習するモード。 |
| `/bookmark` | `BookmarkMode` | 要 | **ブックマーク学習**。ブックマーク済みカードのみを学習。 |
| `/sync-settings` | `SyncSettings` | 要 | デバイス間のクラウド同期設定の詳細画面。 |
| `/diagnostics` | `ImageDiagnostics` | 要 | 画像リンク切れ等の診断・自動修復ツール。 |
