## 2. アプリケーション理解 (1時間)

### 2.1 何を作っているか

**FlashcardMaster** は、効率的な学習を支援する暗記カードアプリです。

**主な特徴**:
- 📱 **オフライン対応**: インターネットがなくても使える
- 🔄 **自動同期**: 複数デバイス間でデータを同期
- 📊 **学習統計**: 学習の進捗を可視化
- 🧠 **SRSアルゴリズム**: 忘却曲線に基づいた復習スケジュール

### 2.2 技術スタック

#### フロントエンド
- **React 18**: UIライブラリ
- **TypeScript**: 型安全な開発
- **Vite**: 高速なビルドツール
- **Tailwind CSS**: ユーティリティファーストCSS

#### データベース
- **IndexedDB** (ローカル): オフライン時のデータ保存
- **Firestore** (クラウド): デバイス間の同期

#### その他
- **Firebase Auth**: ユーザー認証
- **Firebase Storage**: 画像・音声の保存
- **Recharts**: グラフ表示

### 2.3 アーキテクチャ概要

```
┌─────────────┐
│   React UI  │ ← ユーザーが見る画面
└──────┬──────┘
       │
┌──────▼──────┐
│  Services   │ ← ビジネスロジック
│  (TS/JS)    │
└──────┬──────┘
       │
┌──────▼──────────┬──────────────┐
│  IndexedDB      │  Firestore   │ ← データ保存
│  (ローカル)     │  (クラウド)  │
└─────────────────┴──────────────┘
```

**データフロー**:
1. ユーザーが操作 → React UIが反応
2. UIがServiceを呼び出し → ビジネスロジック実行
3. Serviceがデータを保存 → IndexedDB (即座) + Firestore (同期)

### 2.4 主要な画面

| 画面 | 説明 | ファイル |
|------|------|----------|
| ダッシュボード | 今日の復習カード表示 | `Pages/Dashboard.jsx` |
| 学習モード | カードを使った学習 | `Pages/StudyMode.jsx` |
| 統計 | 学習の進捗グラフ | `Pages/Statistics.jsx` |
| カレンダー | 学習履歴カレンダー | `Pages/Calendar.jsx` |
| フォルダ | カードの整理 | `Pages/Folders.jsx` |

### 2.5 重要な概念

#### SRS (Spaced Repetition System)
- 忘却曲線に基づいた復習スケジュール
- カードの「定着度」を計算
- 次回復習日を自動決定

#### オフラインファースト
- ローカル (IndexedDB) が真実の情報源
- クラウド (Firestore) はバックアップ
- 同期は自動で行われる

#### 定着度フェーズ
| フェーズ | 定着度 | 意味 |
|----------|--------|------|
| 要復習 | 0-30% | 忘れかけている |
| 覚えかけ | 30-50% | 少し覚えている |
| 定着途上 | 50-70% | だいぶ覚えている |
| 安定 | 70-85% | しっかり覚えている |
| 長期保持 | 85-100% | 完全に定着 |

## 5. よくある質問

### Q: ビルドエラーが出ました
```
Error: Cannot find module 'xxx'
```

**A**: 依存関係が不足しています
```bash
npm install
```

### Q: TypeScriptエラーが消えません
```
Type 'xxx' is not assignable to type 'yyy'
```

**A**: VSCodeを再起動してください
```
Ctrl+Shift+P → "Reload Window"
```

### Q: データが表示されません

**A**: IndexedDBを確認してください
1. DevTools を開く (F12)
2. Application タブ
3. IndexedDB → flashcardDB
4. データがあるか確認

### Q: ログインできません

**A**: Firebase設定を確認してください
1. `.env` ファイルを確認
2. Firebase Console で設定を確認
3. `firebase login` を再実行

### Q: 画像が表示されません

**A**: Firebase Storageの権限を確認してください
1. Firebase Console → Storage
2. Rules を確認
3. 読み取り権限があるか確認
