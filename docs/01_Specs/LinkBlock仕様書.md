# リンクブロック (Reference Block) 仕様書

## 概要
リンクブロックは、カードの内容に関連する外部リソース（Webサイト、参考文献など）へのリンクを保持するためのブロックです。

## 基本構成
- **URL**: 必須。外部リソースのURL。
- **表示名 (name)**: 任意。URLの代わりに表示されるテキスト。
- **タイプ/説明**: 廃止済み（シンプル化のため）。

## 制約事項
- **1カードにつき最大1個**: 
    一枚のカード（問題側・解答側を合わせた全体）において、リンクブロックは最大で**1つ**までしか追加できません。
    - すでにリンクブロックが存在する場合、ツールバーの「リンク」追加ボタンは無効化（disabled）されます。
    - 外部セクション（問題から解答、またはその逆）への複製時も、この制限が適用されます。

## UI/UX
### 編集モード
- **追加ボタン**: 制限に達している場合は、半透明（opacity-40）になり、クリックできなくなります。
- **複製アクション**: 既にリンクブロックが存在する状態でリンクブロックの複製を試みると、アラートが表示され処理が中断されます。

### 表示モード
- カードの特定のセクション内にインラインで表示されます。
- クリックすると外部ブラウザまたは新しいタブでURLが開きます。

## データ構造
```typescript
type ReferenceBlockData = {
  url: string;
  name?: string;
};

// CardBlock内での利用
{
  id: string,
  type: 'reference',
  references: ReferenceBlockData[], // 現在は1要素のみを想定
  orderIndex: number
}
```

- v1.1: シンプル化（種別・説明を削除、URL/表示名のみに）
