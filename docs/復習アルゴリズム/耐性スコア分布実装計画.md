# 統計画面の分布変更：耐性スコアへの切り替え

## 目的
統計画面の「記憶の定着度分布（Memory Stability）」を、ユーザー仕様に基づく「耐性スコア（Resistance Score）」の分布に変更します。
UIデザイン（棒グラフ、色、グラデーション）は維持し、集計ロジックのみを差し替えます。

## 参照仕様
- `docs/復習アルゴリズム/耐性スコア仕様.md`

## 変更内容

### 1. `src/Components/stats/StatsCharts.tsx`
- **変更前**:
  - `normalizeMemoryStability` を使用して `memoryStability` (0-1) を取得し、×100 して分布を作成。
- **変更後**:
  - `calculateResistanceScore` を使用して、復習間隔に基づくスコア (0-100) を算出。
  - **計算ロジック**:
    - `intervalDays` を算出 (`nextReviewDate` - `lastReviewAt`)。
    - `calculateResistanceScore(intervalDays)` を呼び出す。
- **表示変更**:
  - グラフタイトルを「耐性スコア分布」に変更。
  - 単位や軸のラベル（%）はそのまま使用（スコアも0-100なので適合）。

### 2. `src/utils/reviewMetrics.ts`
- 既存の `calculateResistanceScore` を利用。変更なし。

## 検証計画
- 統計画面を開き、グラフが表示されることを確認。
- タイトルが「耐性スコア分布」になっていることを確認。
- 分布が「復習間隔」に基づいていることを、簡単なデータ（新規、長期など）で確認。
