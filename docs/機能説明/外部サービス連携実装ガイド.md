## 1. Google Drive連携

### 1.1 Google Cloud Consoleでの設定

#### ステップ1: Google Cloudプロジェクトの作成
1. [Google Cloud Console](https://console.cloud.google.com/)にアクセス
2. 新しいプロジェクトを作成（または既存プロジェクトを選択）
3. プロジェクト名を設定（例: "Flash Master Drive Integration"）

#### ステップ2: Google Drive APIの有効化
1. 「APIとサービス」→「ライブラリ」に移動
2. "Google Drive API"を検索
3. 「有効にする」をクリック

#### ステップ3: OAuth同意画面の設定
1. 「APIとサービス」→「OAuth同意画面」に移動
2. ユーザータイプを選択（外部ユーザーの場合は承認が必要）
3. アプリ情報を入力：
   - アプリ名: Flash Master
   - ユーザーサポートメール: あなたのメールアドレス
   - 開発者の連絡先情報: あなたのメールアドレス
4. スコープを追加：
   - `https://www.googleapis.com/auth/drive.file`（ファイルの作成・編集・削除）
   - `https://www.googleapis.com/auth/drive.readonly`（読み取り専用）
5. テストユーザーを追加（開発中の場合）

#### ステップ4: 認証情報の作成
1. 「APIとサービス」→「認証情報」に移動
2. 「認証情報を作成」→「OAuth 2.0 クライアント ID」を選択
3. アプリケーションの種類: 「ウェブアプリケーション」
4. 名前を設定（例: "Flash Master Web Client"）
5. 承認済みの JavaScript 生成元:
   - 開発環境: `http://localhost:5173`
   - 本番環境: `https://your-domain.com`
6. 承認済みのリダイレクト URI:
   - 開発環境: `http://localhost:5173/auth/google/callback`
   - 本番環境: `https://your-domain.com/auth/google/callback`
7. 「作成」をクリック
8. **クライアント ID**と**クライアントシークレット**をメモ（後で使用）
 299036595781-9893a8tdhv07081hqjr1mtmgaaogvkci.apps.googleusercontent.com
 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

`.env`ファイルに追加：


### 1.3 実装手順

#### 必要なパッケージのインストール

```bash
npm install @react-oauth/google googleapis
```

#### Firebase Cloud Functionsでの実装

`functions/index.js`に追加：  ok

```javascript
const { onCall } = require("firebase-functions/v2/https");
const { google } = require("googleapis");
const admin = require("firebase-admin");

// Google Drive API クライアントの作成
async function getDriveClient(accessToken) {
  const auth = new google.auth.OAuth2();
  auth.setCredentials({ access_token: accessToken });
  return google.drive({ version: 'v3', auth });
}

// フォルダをGoogle Driveにエクスポート
exports.exportToGoogleDrive = onCall(
  { region: 'asia-northeast1' },
  async (request) => {
    const { auth, data } = request;
    if (!auth) {
      throw new HttpsError("unauthenticated", "認証が必要です");
    }

    const { folderId, accessToken } = data;
    const drive = await getDriveClient(accessToken);

    // フォルダとカードを取得
    const folderDoc = await db.collection('folders').doc(folderId).get();
    const cardsSnapshot = await db.collection('cards')
      .where('folderId', '==', folderId)
      .get();

    // Google Driveにフォルダを作成
    const folderResponse = await drive.files.create({
      requestBody: {
        name: folderDoc.data().name,
        mimeType: 'application/vnd.google-apps.folder',
      },
    });

    // カードをJSONファイルとしてアップロード
    const cardsData = cardsSnapshot.docs.map(doc => doc.data());
    await drive.files.create({
      requestBody: {
        name: 'cards.json',
        parents: [folderResponse.data.id],
      },
      media: {
        mimeType: 'application/json',
        body: JSON.stringify(cardsData, null, 2),
      },
    });

    return { success: true, folderId: folderResponse.data.id };
  }
);
```

```javascript
exports.exportToDropbox = onCall(
  { region: 'asia-northeast1' },
  async (request) => {
    const { auth, data } = request;
    if (!auth) {
      throw new HttpsError("unauthenticated", "認証が必要です");
    }

    const { folderId, accessToken } = data;
    const dbx = new Dropbox({ accessToken });

    // フォルダとカードを取得
    const folderDoc = await db.collection('folders').doc(folderId).get();
    const cardsSnapshot = await db.collection('cards')
      .where('folderId', '==', folderId)

    const jsonContent = JSON.stringify(cardsData, null, 2);

    // Dropboxにアップロード
    await dbx.filesUpload({
      path: `/${folderDoc.data().name}/cards.json`,
      contents: jsonContent,
    });

    return { success: true };
  }
);
```

#### 必要なパッケージのインストール

```bash
npm install @microsoft/microsoft-graph-client @azure/msal-browser
```

#### Firebase Cloud Functionsでの実装

```javascript
const { Client } = require('@microsoft/microsoft-graph-client');

exports.exportToOneDrive = onCall(
  { region: 'asia-northeast1' },
  async (request) => {
    const { auth, data } = request;
    if (!auth) {
      throw new HttpsError("unauthenticated", "認証が必要です");
    }

    const { folderId, accessToken } = data;
    const client = Client.init({
      authProvider: (done) => {
        done(null, accessToken);
      },
    });

    // フォルダとカードを取得
    const folderDoc = await db.collection('folders').doc(folderId).get();
    const cardsSnapshot = await db.collection('cards')
      .where('folderId', '==', folderId)
      .get();

    // カードデータをJSONに変換
    const cardsData = cardsSnapshot.docs.map(doc => doc.data());
    const jsonContent = JSON.stringify(cardsData, null, 2);

    // OneDriveにアップロード
    await client.api('/me/drive/root:/FlashMaster/' + folderDoc.data().name + '/cards.json:/content')
      .put(jsonContent);

    return { success: true };
  }
);
```

#### ステップ1: Webhookエンドポイントの作成
Firebase Cloud FunctionsでWebhookエンドポイントを作成：

```javascript
const stripe = require('stripe')(functions.config().stripe.secret_key);

exports.stripeWebhook = onRequest(
  { region: 'asia-northeast1' },
  async (req, res) => {
    const sig = req.headers['stripe-signature'];
    let event;

    try {
      event = stripe.webhooks.constructEvent(
        req.rawBody,
        sig,
        functions.config().stripe.webhook_secret
      );
    } catch (err) {
      res.status(400).send(`Webhook Error: ${err.message}`);
      return;
    }

    // イベントタイプに応じた処理
    switch (event.type) {
      case 'checkout.session.completed':
        const session = event.data.object;
        // サブスクリプションを有効化
        await activateSubscription(session.customer_email, session.subscription);
        break;
      case 'customer.subscription.deleted':
        // サブスクリプションを無効化
        await deactivateSubscription(event.data.object.customer);
        break;
      // その他のイベントタイプ...
    }

    res.json({ received: true });
  }
);
```

```javascript
const stripe = require('stripe')(functions.config().stripe.secret_key);

// サブスクリプション作成
exports.createSubscription = onCall(
  { region: 'asia-northeast1' },
  async (request) => {
    const { auth, data } = request;
    if (!auth) {
      throw new HttpsError("unauthenticated", "認証が必要です");
    }

    const { priceId } = data;
    const userId = auth.uid;

    // ユーザードキュメントを取得
    const userDoc = await db.collection('users').doc(userId).get();
    let customerId = userDoc.data()?.stripeCustomerId;

    // Stripe顧客を作成（存在しない場合）
    if (!customerId) {
      const customer = await stripe.customers.create({
        email: auth.token.email,
        metadata: { userId },
      });
      customerId = customer.id;
      await db.collection('users').doc(userId).update({
        stripeCustomerId: customerId,
      });
    }

    // チェックアウトセッションを作成
    const session = await stripe.checkout.sessions.create({
      customer: customerId,
      mode: 'subscription',
      payment_method_types: ['card'],
      line_items: [{ price: priceId, quantity: 1 }],
      success_url: 'https://your-domain.com/settings?success=true',
      cancel_url: 'https://your-domain.com/settings?canceled=true',
      metadata: { userId },
    });

    return { sessionId: session.id };
  }
);

// プラン制限チェック
exports.checkPlanLimit = onCall(
  { region: 'asia-northeast1' },
  async (request) => {
    const { auth } = request;
    if (!auth) {
      throw new HttpsError("unauthenticated", "認証が必要です");
    }

    const userId = auth.uid;
    const userDoc = await db.collection('users').doc(userId).get();
    const userData = userDoc.data();
    const plan = userData?.subscriptionPlan || 'free';

    // プランに応じた制限を返す
    const limits = {
      free: { maxCards: 50, maxFolders: 10, maxStorage: 100 }, // MB
      pro: { maxCards: Infinity, maxFolders: Infinity, maxStorage: 10000 }, // MB
    };

    return limits[plan] || limits.free;
  }
);
```

#### フロントエンドでの実装 ok

`src/services/stripeService.ts`を作成：

```typescript
import { loadStripe } from '@stripe/stripe-js';
import { httpsCallable } from 'firebase/functions';
import { functions } from './firebase';

const stripePromise = loadStripe(import.meta.env.VITE_STRIPE_PUBLISHABLE_KEY);

export async function createSubscription(priceId: string) {
  const createSubscriptionFunction = httpsCallable(functions, 'createSubscription');
  const result = await createSubscriptionFunction({ priceId });
  const { sessionId } = result.data as { sessionId: string };

  const stripe = await stripePromise;
  if (stripe) {
    await stripe.redirectToCheckout({ sessionId });
  }
}
```

#### ステップ1: Cloud Scheduler APIの有効化
1. [Google Cloud Console](https://console.cloud.google.com/)にアクセス
2. 「APIとサービス」→「ライブラリ」に移動
3. "Cloud Scheduler API"を検索して有効化

#### ステップ2: サービスアカウントの作成
1. 「IAMと管理」→「サービスアカウント」に移動
2. 「サービスアカウントを作成」をクリック
3. 名前を入力（例: "flash-master-scheduler"）
4. 「作成して続行」をクリック
5. ロール: 「Cloud Functions 管理者」を選択
6. 「完了」をクリック

#### ステップ3: スケジュールジョブの作成
1. 「Cloud Scheduler」に移動
2. 「ジョブを作成」をクリック
3. ジョブ情報を入力：
   - 名前: daily-notification
   - リージョン: asia-northeast1
   - 頻度: `0 9 * * *`（毎日9時）
   - タイムゾーン: Asia/Tokyo
   - ターゲットタイプ: HTTP
   - URL: `https://asia-northeast1-your-project.cloudfunctions.net/sendDailyNotifications`
   - HTTPメソッド: POST
   - 認証ヘッダー: OIDC トークンを追加
   - サービスアカウント: 作成したサービスアカウントを選択

### 7.2 Firebase Cloud Functionsでの実装

`functions/index.js`に追加：

```javascript
const { onSchedule } = require("firebase-functions/v2/scheduler");
const admin = require("firebase-admin");

// 毎日の通知送信
exports.sendDailyNotifications = onSchedule(
  {
    schedule: "0 9 * * *", // 毎日9時（JST）
    timeZone: "Asia/Tokyo",
    region: "asia-northeast1",
  },
  async (event) => {
    const today = new Date().toISOString().split('T')[0];
    
    // 通知を送信すべきユーザーを取得
    const usersSnapshot = await db.collection('userSettings')
      .where('notificationsEnabled', '==', true)
      .get();

    for (const userDoc of usersSnapshot.docs) {
      const userId = userDoc.id;
      const settings = userDoc.data();

      // 今日復習すべきカード数を取得
      const cardsSnapshot = await db.collection('cards')
        .where('userId', '==', userId)
        .where('nextReviewDate', '<=', admin.firestore.Timestamp.fromDate(new Date()))
        .get();

      const reviewCount = cardsSnapshot.size;

      if (reviewCount > 0) {
        // ブラウザプッシュ通知を送信
        await sendBrowserNotification(userId, {
          title: 'Flash Master',
          body: `今日は${reviewCount}枚のカードを復習しましょう！`,
          icon: '/icon-192.png',
          badge: '/icon-192.png',
          data: {
            url: '/',
          },
        });
      }
    }

    logger.info('Daily notifications sent');
  }
);

// ブラウザプッシュ通知を送信
async function sendBrowserNotification(userId, notification) {
  const tokensSnapshot = await db.collection('userNotificationTokens')
    .where('userId', '==', userId)
    .get();

  if (tokensSnapshot.empty) return;

  const tokens = tokensSnapshot.docs.map(doc => doc.data().token);
  
  // Firebase Cloud Messagingを使用
  const messaging = admin.messaging();
  await messaging.sendMulticast({
    tokens,
    notification,
    webpush: {
      notification: {
        ...notification,
        requireInteraction: true,
      },
    },
  });
}
```

### 7.3 フロントエンドでの実装

通知許可のリクエストとトークンの保存：

```typescript
import { getMessaging, getToken, onMessage } from 'firebase/messaging';
import { messaging } from './firebase';

export async function requestNotificationPermission() {
  const permission = await Notification.requestPermission();
  if (permission === 'granted') {
    const token = await getToken(messaging, {
      vapidKey: import.meta.env.VITE_FIREBASE_VAPID_KEY,
    });
    
    // トークンをFirestoreに保存
    await setDoc(doc(db, 'userNotificationTokens', token), {
      userId: currentUser.uid,
      token,
      createdAt: serverTimestamp(),
    });
    
    return token;
  }
  return null;
}
```

## まとめ

各外部サービス連携の実装には以下の共通ステップがあります：

1. **サービスアカウントの作成・登録**
2. **APIキー・認証情報の取得**
3. **環境変数の設定**
4. **Firebase Cloud Functionsでの実装**（サーバー側処理）
5. **フロントエンドでの実装**（クライアント側処理）
6. **テスト・デバッグ**
7. **本番環境へのデプロイ**

**重要な注意事項**:
- APIキーやシークレットはクライアント側に公開しないこと
- 本番環境では適切なエラーハンドリングを実装すること
- 各サービスの利用規約を確認すること
- レート制限に注意すること
- セキュリティベストプラクティスに従うこと

実装に際しては、各サービスの最新ドキュメントを参照することを推奨します。
