# Map機能拡張 実装計画

## 1. 概要
現在のMap機能（局所地図、世界地図）に対し、仕様書との乖離を埋めるための機能強化を行います。具体的には、タグ色の統合、簡易AI補助、初回体験（チュートリアル）、力学モデルの改善、孤立ノードの強調表示を実装します。

## 2. 変更内容の詳細

### 2.1 タグ色の完全統合
- **現状**: `WorldMap` は文字列ハッシュ、`LocalMap` は固定色を使用。
- **変更**:
    - `useTags` フックを利用して、正しいタグ色を取得するように変更。
    - `LocalMap.tsx` と `WorldMap.tsx` のノード描画ロジックを更新。
    - カードにタグがない場合はデフォルト色（白/グレー）を使用。
    - 複数のタグがある場合は、先頭のタグの色を優先（仕様通り）。

### 2.2 AI補助機能（簡易実装）
- **方針**: サーバーサイドAIではなく、ローカルの情報を活用した「関連カード推薦」を実装。
- **ロジック**:
    - 共通のタグを持つカードを「関連度が高い」と判定。
    - 同じフォルダ内のカードを推奨。
- **UI**:
    - `HandTray` に「おすすめ（Recommended）」タブ、またはハイライト機能を追加。

### 2.3 初回体験アニメーション（Onboarding）
- **機能**:
    - 初めてMapを開いたユーザーに対し、操作ガイドを表示。
    - 手順: 「カードをドラッグして配置」→「ノード同士を近づけて接続」。
- **実装**:
    - `useLocalStorage` でチュートリアル完了フラグを管理。
    - オーバーレイガイドコンポーネント (`MapTutorialOverlay`) を新規作成。

### 2.4 線の自動最適化（Graph Layout）
- **課題**: ノードが増えると線が重なりやすい。
- **対策**:
    - `d3-force` のパラメータ調整。
    - `forceLink` の距離 (`distance`) を動的に調整。
    - `forceCollide` （衝突判定）の半径をノードサイズに合わせて最適化。

### 2.5 未接続カードの点滅（孤立ノード強調）
- **仕様**: 局所地図でつながっていないカードを目立たせる。
- **実装**:
    - `LocalMap` のノード描画時に、リンクを持たないノードを検出。
    - CSSアニメーション（opacityの点滅や枠線の発光）を適用。
