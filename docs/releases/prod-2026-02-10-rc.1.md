# prod-2026-02-10-rc.1 Release Notes

Date: 2026-02-10

## Summary

This release bundles PDF zoom confinement hardening, PPTX converter operational readiness, and repository operation guardrails.

## PDF

- Confined zoom input handling to the PDF viewer region (Ctrl/Cmd+Wheel and pinch paths).
- Reduced parent scroll-chain side effects for PDF interactions.
- Added regression coverage for PDF zoom scale logic.

## PPTX

- Added converter service integration and conversion guards in Functions.
- Added deployment and operational scripts for converter setup and predeploy validation.
- Added E2E scripts for success/failure converter scenarios.
- Added converter operations runbook documentation.

## Operations

- Added commit hygiene guidance to reduce staged mix-in accidents.
- Added QA docs index and standard release QA references.
- Standardized TypeScript validation command via `npm run typecheck`.

## Known Issues

- Existing build chunk-size warnings remain (Vite/Rollup warning output).
- Existing dynamic-import chunk warnings remain.
- These warnings are pre-existing and are not introduced by this release bundle.
