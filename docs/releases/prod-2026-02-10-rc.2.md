# prod-2026-02-10-rc.2 Release Notes

Date: 2026-02-10

## Summary

Post-`rc.1` fixes for PPTX converter runtime behavior and E2E operations scripts.

## Changes

- Updated PPTX converter runbook with explicit credentials usage for E2E scripts.
- Refined converter ID token audience handling in `functions/src/pptxConversion.ts`:
  - Uses endpoint origin as audience for ID token acquisition and request headers.
- Enhanced E2E scripts:
  - Added `-CredentialsFile` option to `scripts/e2e-pptx-real.ps1` and `scripts/e2e-pptx-failure.ps1`.
  - Added credential resolution order:
    1. `-CredentialsFile`
    2. `GOOGLE_APPLICATION_CREDENTIALS`
    3. `./serviceAccountKey.json`
    4. ADC fallback
  - Added bucket access preflight checks via `gsutil ls`.
