# XLSXテンプレートダウンロード機能の実装計画

## 概要
ユーザーがバルクインポート用の正しい形式を知るためのXLSXテンプレートをダウンロードできるようにします。

## 実施事項

### 1. テンプレートダウンロードロジックの作成
- ファイルパス: `src/features/import/xlsx/downloadXlsxImportTemplate.ts`
- 内容:
    - `xlsx` ライブラリを使用して、サンプルデータを含むワークブックを作成します。
    - `blocks` シートと `readme` シートを含めます。
    - 列幅の調整を行い、読みやすくします。
    - ブラウザでダウンロードを実行します。

### 2. インポートダイアログのUI修正
- ファイルパス: `src/features/import/ui/XlsxImportDialog.tsx`
- 内容:
    - `downloadXlsxImportTemplate` をインポートします。
    - テンプレートダウンロード用のボタンを追加します。
    - ダイアログを閉じる際の状態リセット処理を確実に実行するように修正します。
    - ファイル選択エリアのラベル横にダウンロードボタンを配置します。
    - インポートに関する補足説明を追加します。

## 影響範囲
- インポートダイアログのUI
- `xlsx` ライブラリへの依存（既に使用されている想定）

## 確認事項
- [ ] テンプレートが正しくダウンロードされること
- [ ] ダウンロードされたファイルの内容（シート名、ヘッダー、サンプル）が正しいこと
- [ ] インポート実行中や解析中にダウンロードボタンが適切に無効化されること
