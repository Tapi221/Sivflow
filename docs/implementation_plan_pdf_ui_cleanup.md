# 実装プラン: PDF ビューア UI のクリーンアップ

## 概要
PDF ビューアのツールバーにおいて、表示内容を整理し、よりコンパクトで実用的な UI に改善します。具体的には、重複するドキュメント名の表示を削除し、同期ステータスの表示を最適化します。

## 変更内容

### 1. PdfPane.tsx の変更
- ツールバーに渡していた `displayName` プロパティに関連するロジックを削除します。

### 2. PdfPaneToolbar.tsx の変更
- `PdfPaneToolbarProps` から `displayName` を削除します。
- `displayName` の表示（タイトルと「PDF」というラベル）を削除します。
- ローカルのみの警告や同期失敗のエラーメッセージのレイアウトを調整し、メッセージがない場合はスペースを空けるように変更します。
- ツールバー右側の操作ボタン群が潰れないように `shrink-0` を追加します。

## 影響範囲
- `src/components/pdf/PdfPane.tsx`
- `src/components/pdf/PdfPaneToolbar.tsx`

## 確認事項
- [ ] ツールバーからドキュメント名の冗長な表示が消えているか。
- [ ] ローカルのみの場合や失敗時にステータス文言が正しく表示されるか。
- [ ] ツールバー右側のボタン群のレイアウトが崩れていないか。
