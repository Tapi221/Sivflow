# 実装計画：作成モード選択UIの改善

「作成モード選択」画面のUIを、設定画面やフォルダ作成画面などの他の洗練されたUIのデザインに合わせ、アプリ全体の統一感とプレミアム感を高めます。

## ユーザーレビューが必要な項目

> [!IMPORTANT]
> - アイコンの配色は、各モードの視認性を高めるために現在の系統（青、緑、紫、オレンジ）を維持しつつ、トーンを調整します。
> - ダイアログのサイズや角丸の半径（`rounded-2xl`）を他の最新のダイアログと統一します。

## 変更内容

### UI コンポーネント

#### [MODIFY] src/Components/card/CreationModeDialog.jsx

- **ダイアログ全体の調整**:
    - `DialogContent` の `className` を `sm:max-w-2xl rounded-2xl md:rounded-xl p-5 md:p-6 gap-5 md:gap-6` に更新し、モバイルでの表示を最適化します。
    - 影（`shadow-xl`）の適用。
- **ヘッダー部分**:
    - 戻るボタンのスタイルを調整し、隣接するテキストとのバランスを改善します。
    - `DialogTitle`, `DialogDescription` のフォントサイズとウェイトを調整します。
- **モード選択カード (グリッドアイテム)**:
    - 各項目の背景色を `bg-white` または `bg-[#F8FAFB]` にし、ボーダーを `border-slate-100` に設定します。
    - ホバー時のエフェクトを強化（`hover:border-primary-200`, `hover:bg-primary-50/30`, `hover:shadow-md`）。
    - アイコンコンテナの角丸とパディングを調整し、アイコンの視認性を向上させます。
    - タイトルと説明文のタイポグラフィを改善します。
