# 実装計画：フォルダツリードラッグハンドルの配置調整

フォルダツリーのドラッグハンドル（並び替えボタン）を、より左端に寄せるための調整を行います。

## 課題
- 現在、ドラッグハンドルとカードの左端の間に余白があり、ユーザーはこれをより左に寄せることを希望している。

## 変更内容

### [FolderTree](file:///c:/FlashcardMaster/src/Components/folder/FolderTree.tsx)
- `FolderItem` のコンテナパディングを調整します。
    - 現状: `px-1.5 md:px-2`
    - 修正案: `pl-0.5 pr-1.5 md:pl-1 md:pr-2` に変更し、左側のパディングを最小限にします。
- 必要に応じて、ドラッグハンドル自体のマージンを微調整します。

## 検証計画
- [ ] フォルダアイテムの左端の余白が減り、ドラッグハンドルがより左側に配置されていることを確認。
- [ ] アイコンがカードの角丸部分に重なりすぎて不自然になっていないか確認。
- [ ] モバイルとデスクトップの両方でレイアウトが適切であることを確認。
