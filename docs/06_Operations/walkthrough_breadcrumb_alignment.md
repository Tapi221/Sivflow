# 完了報告：フォルダー画面パンくずリストの右寄せ

フォルダー画面のヘッダーレイアウトを調整し、パンくずリストを右側に配置しました。

## 変更内容

### [FolderView](file:///c:/FlashcardMaster/src/Pages/FolderView.jsx)
- **ヘッダーレイアウトの刷新**: 
    - 左側には「戻るボタン」のみを配置し、視認性を高めました。
    - 右側のアクションエリア（新規カード、学習開始ボタンなど）の左隣にパンくずリストを移動しました。
- **右寄せの実装**: 
    - パンくずリストのコンテナに `justify-end` を適用し、画面右側に吸い付くような配置にしました。
    - パンくずリストが長くなった場合でも、アクションボタンを押し出さないように `flex-1` と `min-w-0` で適切に領域を制御しています。
- **レスポンシブ対応**: 
    - パンくずの各要素の最大幅（`max-w-[120px]`）を調整し、モバイル端末でもボタン類と共存できるように最適化しました。

- [x] 長いフォルダー名や深い階層でもレイアウトが崩れず、省略表示が機能することを確認。
