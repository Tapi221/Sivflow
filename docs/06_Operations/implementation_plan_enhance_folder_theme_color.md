# フォルダ一覧のテーマカラー強化

## 概要
「テーマカラーをもう少し入れて」という要望に応え、フォルダ管理画面において、アプリのアクセントカラー（プライマリカラー）を効果的に配置し、デザインのアイデンティティを強めます。これまでの「コンパクト化」を維持しつつ、色による識別性と華やかさを向上させます。

## 変更内容

### Components/folder

#### [MODIFY] [FolderTree.tsx](file:///c:/FlashcardMaster/src/Components/folder/FolderTree.tsx)

- [x] **フォルダアイコン**:
  - 色を `text-primary-600` に変更し、背景チップに `bg-primary-50/50` を追加しました。

- [x] **メタデータ (カード数) の強調**:
  - `text-primary-700/70` かつ `font-bold` に変更しました。

- [x] **選択状態の強化**:
  - 選択時の背景を `bg-primary-50/60` に変更。
  - ホバー時も `hover:bg-primary-50/30` のテーマカラーが適用されます。

- [x] **通知OFFアイコン**:
  - アイコン色を `text-amber-500/60` に調整しました。

- [x] **セパレーター**:
  - `•` の色を `text-primary-200` (未学習時) に変更しました。

## 検証計画
- 画面全体を見渡した時に、プライマリカラーがバランスよく散りばめられているか確認。
- 視認性が低下していないか確認（特に背景色と文字色のコントラスト）。
- 他の画面（ダッシュボード等）との一貫性を確認。
