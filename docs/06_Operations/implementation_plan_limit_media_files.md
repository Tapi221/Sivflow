# 実装計画：画像ブロックの画像数制限

画像ブロックにおいて、1つのブロックにつき画像は1枚までという制限を設けます。

## 変更内容

### `src/Components/card/blocks/MediaBlock.tsx`

`MediaUploader` に渡す `maxFiles` プロパティを、現在の `10`（画像の場合）から `1` に変更します。
音声ブロックは既に `1` に設定されているため、結果的にメディアブロック（画像・音声）はすべて「1ブロックにつき1ファイル」の仕様になります。

## 検証計画

- [ ] **画像枚数制限の確認**:
    - 画像ブロックを作成し、画像を1枚追加する。
    - 1枚追加した状態で、さらに画像を追加しようとする（ドラッグ＆ドロップ、またはファイル選択）。
    - 追加のアクションが制限される（「追加」ボタンが消える、またはドロップしても反応しない）ことを確認する。
- [ ] **既存動作への影響**:
    - 音声ブロックの動作（既に1枚制限）に変更がないことを確認する。
    - 既存のブロックで既に複数枚の画像がある場合、表示が崩れないことを確認する（ただし、新規追加はできなくなる）。
