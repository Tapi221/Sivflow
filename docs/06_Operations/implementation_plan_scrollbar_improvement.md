# スクロールバーのデザイン改善計画

スクロールバーの現在のデザイン（黒い縁取り）を削除し、よりモダンで洗練されたデザインに変更します。

## 変更内容

### [UI改善] スクロールバーのスタイル変更

`src/index.css` に定義されているスクロールバーのスタイルを修正します。

#### [MODIFY] [index.css](file:///c:/FlashcardMaster/src/index.css)

- `::-webkit-scrollbar` の幅を 8px から 6px に変更し、スリムにします。
- `::-webkit-scrollbar-track` の背景を透明（または非常に薄い色）にし、ボーダーを削除します。
- `::-webkit-scrollbar-thumb` のボーダーを削除し、角丸を大きくして「ピル型」にします。
- Firefox 対応のために `scrollbar-width` と `scrollbar-color` も設定します。

- [ ] スクロール操作に支障がないか確認する。
