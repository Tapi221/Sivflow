# テキストブロック表示同期の実装完了

## 概要
カード編集画面のテキストボックスの表示を、閲覧画面（Flashcard Viewer）と一致させました。
これにより、改行位置や文字サイズが編集時と閲覧時で同じになり、「編集画面では改行されていないのに閲覧画面では改行されている」といった違和感を解消しました。

## 実装内容
`src/Components/card/blocks/TextBlock.tsx` を修正しました。

*   **フォント**: `font-serif`（明朝体/セリフ体）に変更し、閲覧画面と統一。
*   **フォントサイズ**: レスポンシブ対応の `clamp` 関数と `md:text-2xl` を適用。
*   **最大幅**: `max-w-2xl` (約672px) を設定し、閲覧画面と同じ幅で折り返すように制限。
*   **配置**: `mx-auto` で中央寄せ。
*   **行間**: `leading-relaxed` で少し広めの行間に統一。
*   **折り返し**: `break-all` を適用し、単語の途中でも折り返す挙動を統一（日本語テキスト向け）。

## 検証方法
1.  カード編集画面を開きます。
2.  長いテキストを入力し、特定の位置で自動改行されることを確認します。
3.  カードを保存して裏返す（またはプレビューする）と、編集画面と全く同じ位置で改行されていることを確認できます。
