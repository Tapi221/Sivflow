# コードブロックのスタイリング刷新（Claude風）

コードブロックの見た目を、Claudeのように「締まったサイズ感」と「高い視認性」を両立したデザインに刷新します。問題文（本文）を引き立てつつ、コード領域も機能的で馴染みの良いものにします。

## 修正内容

### [CodeRenderer.tsx](file:///c:/FlashcardMaster/src/Components/card/CodeRenderer.tsx)

#### 1. 外面デザインの統合
- **余白の縮小**: `px-4 py-3` に固定。1行コードでも間延びせず、複数行でも締まって見える密度にします。
- **角丸と枠線**: `rounded-xl` を採用し、紙の質感に馴染む薄い枠線を維持します。
- **背景色**: 既存のテーマカラーを維持しつつ、密度感を高めます。

#### 2. 言語ラベルの刷新（左上配置）
- **バッジの廃止**: 右側の丸い "JS" バッジを廃止します。
- **左上配置**: コードの冒頭（左上）に小さく絶対配置 (`absolute top-1.5 left-3`) します。
- **スタイル**: `text-[11px]`, 薄い背景、透過度を調整し、コードの読みやすさを邪魔しない控えめなデザインにします。コード自体の開始位置（padding）をラベルと重ならないよう微調整します。

#### 3. タイポグラフィ
- **フォントサイズ**: 13px 〜 14px 相当 (`text-[13.5px]` など調整)。
- **行間**: `leading-5` (20px) に固定。

### [CodeBlockEditor.tsx](file:///c:/FlashcardMaster/src/Components/card/CodeBlockEditor.tsx)

#### 1. 編集UIの同期
- 言語選択のドロップダウンを左上に移動し、Viewモードのラベル位置と同期させます。
- タイポグラフィ設定（14px / 20px）をViewと完全に一致させます。

### [index.css](file:///c:/FlashcardMaster/src/index.css)

#### 1. グローバルスタイルの調整
- `.codeBlockPre` の `padding-right`（以前のバッジ用余白）を削除し、左右対称な余白に直します。

## 検証計画

### 手動確認
1.  **表示モード**:
    *   1行コードがコンパクトに表示されるか。
    *   左上の言語ラベルがコードの最初の単語と重なりすぎていないか。
    *   モバイル幅でラベルが邪魔になっていないか。
2.  **編集モード**:
    *   入力中に左上の言語選択が操作しやすいか。
    *   表示モードとのフォントサイズ・行間の差異がないか。

### 自動検証
- `npm run build` での正常終了を確認。
