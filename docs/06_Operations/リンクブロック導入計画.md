# リンクブロック (ReferenceBlock) 導入計画

カードに補足情報として外部リンク（参考資料）を添付できる「リンクブロック」を導入します。これは学習の質を高めるための「リファレンス」としての位置付けであり、学習の集中を妨げない控えめな表示と、必要な時に素早くアクセスできる操作性を両立させます。

## Proposed Changes

### 1. 型定義の拡張
- [MODIFY] [index.ts](file:///c:/FlashcardMaster/src/types/index.ts)
    - `ReferenceBlockData` 型を新規定義（id, url, name, type, description）。
    - `CardBlock` の `type` に `'reference'` を追加し、`references` プロパティを追加。

### 2. データ処理ロジックの更新
- [MODIFY] [index.ts](file:///c:/FlashcardMaster/src/utils/index.ts)
    - `normalizeCard`: `reference` ブロックの正規化処理を追加。
- [MODIFY] [localDB.ts](file:///c:/FlashcardMaster/src/services/localDB.ts)
    - `denormalizeCardForStorage`: リンクブロックからのテキスト抽出（必要に応じて）などを考慮。

### 3. 学習画面 (Learning Screen) UIの実装
- [MODIFY] [Flashcard.tsx](file:///c:/FlashcardMaster/src/Components/card/Flashcard.tsx)
    - カード右下に「📎 x N」形式のインジケータを表示。
    - インジケータをクリックした際に、リンク一覧を表示する `BottomSheet` または `Dialog` を表示。
    - リンク一覧から1タップで外部ブラウザへ遷移（ポップアップ内で詳細を確認可能）。

### 4. エディタ画面 (Editor Screen) UIの実装
- [NEW] `src/Components/card/blocks/ReferenceBlock.tsx`
    - リンクのタイトル、URL、種別、説明を編集するためのブロックコンポーネント。
- [MODIFY] [BlockEditor.tsx](file:///c:/FlashcardMaster/src/Components/card/BlockEditor.tsx)
    - ブロック追加ツールバーに「リンク」ボタンを追加。
    - `ReferenceBlock` をリストレンダリング対象に追加。
