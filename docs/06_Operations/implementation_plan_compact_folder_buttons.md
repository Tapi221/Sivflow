# フォルダ一覧のボタン間隔縮小

## 概要
ユーザーからの要望「並び替えボタンと折りたたみボタンの間の余白をできる限りなくして」に対応するため、これらのコントロールを専用のコンテナにまとめ、間隔を最小化（0または0.5）にします。

## 変更内容

### Components/folder

#### [MODIFY] [FolderTree.tsx](file:///c:/FlashcardMaster/src/Components/folder/FolderTree.tsx)

- [x] **コントロールのグルーピング**:
  - `GripVerticalIcon` (または `Checkbox`) と「折りたたみ/展開ボタン」を新しい `div` でラップし、`gap-0` を適用しました。

- [x] **Checkbox マージンの削除**:
  - `Checkbox` の `mr-1` を削除し、グルーピング側の `gap-0` で制御するようにしました。

- [x] **全体のバランス微調整**:
  - アイコンサイズを `w-3.5 h-3.5` に微調整し、全体的にタイトな配置にしました。

## 検証計画
- ローカルサーバーで表示確認。
- グリップと折りたたみボタンが極限まで近づいていることを確認。
- 折りたたみボタンのクリックがグリップのドラッグと干渉しないか確認。
