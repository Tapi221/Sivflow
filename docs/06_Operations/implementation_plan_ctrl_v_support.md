# 実装計画：画像・音声ブロックのCtrl+V対応とショートカット説明追加

画像ブロックおよび音声ブロックにおいて、Ctrl+V（貼り付け）によるアップロードが可能であることを明示し、実際に音声ブロックでも貼り付けができるように実装を修正します。また、設定画面のショートカットキー一覧にもこの機能を追加します。

## 変更内容

### 1. `src/Components/card/MediaUploader.tsx`

- **音声ブロックの貼り付け対応**:
    - `handlePaste` 関数内の `if (type !== 'image') return;` を緩和し、音声ファイル（`audio/*`）の貼り付けも受け付けるように修正します。
- **UIテキストの更新**:
    - アップロードエリアのテキスト「ドラッグ＆ドロップまたはクリックして...アップロード」を、「ドラッグ＆ドロップ、クリック、または **Ctrl+V** で...アップロード」のように更新し、キーボード操作が可能であることを明示します。

### 2. `src/Components/settings/SettingsDialog.jsx`

- **ショートカット一覧の追加**:
    - 「ショートカット」タブの「カードエディタ」セクションに、以下の項目を追加します。
        - キー: `Ctrl + V`
        - 説明: 画像または音声を貼り付け（該当ブロックへのホバー時）
