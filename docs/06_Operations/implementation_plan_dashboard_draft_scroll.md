# 実装計画：ダッシュボード「作成中のカード」の横スクロール最適化

ダッシュボードの「作成中のカード（Drafts）」セクションにおいて、カードが多数ある場合に横スクロールで閲覧できるように最適化します。

## 課題
- 現状 `overflow-x-auto` が指定されているが、`scrollbar-hide` クラスが未定義のため意図した表示になっていない可能性がある。
- スクロール可能であることが視覚的に分かりにくい可能性がある。

## 変更内容

### [Dashboard](file:///c:/FlashcardMaster/src/Pages/Dashboard.jsx)
- `scrollbar-hide` を `no-scrollbar`（`index.css` で定義済み）に変更。
- 横スクロールコンテナに `mask-gradient-right`（`index.css` で定義済み）を適用し、右側に続きがあることを視覚的に示す。
- モバイル表示時において、スクロール領域が適切に確保されるように調整。

## 検証計画
- [ ] 作成中のカードを複数作成し、ダッシュボードで横スクロールができることを確認。
- [ ] スクロールバーが表示されず、かつスクロール操作が可能であることを確認。
- [ ] 右側のフェードアウト効果（マスク）が正しく表示されていることを確認。
- [ ] デスクトップとモバイルの両方でレイアウトが崩れていないことを確認。
