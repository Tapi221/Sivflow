# ダッシュボードのデザイン改善（ステータス表示のバッジ化とレイアウト最適化）

ダッシュボードの見出しの右側にステータス項目を移動し、デザインをより魅力的に（可愛らしく）変更します。

## ユーザーレビューが必要な項目
> [!IMPORTANT]
> ステータス表示を横並びに配置するため、画面幅が狭いモバイル端末では折り返しが発生する可能性があります。レスポンシブ対応を慎重に行います。

## 変更内容

### Dashboard Component

#### [MODIFY] [FolderDashboard.tsx](file:///c:/FlashcardMaster/src/Components/folder/FolderDashboard.tsx)
- `lucide-react` からアイコン（`Files`, `CalendarCheck`, `HelpCircle`, `History`）をインポート。
- ヘッダー部分を `flex` レイアウトに変更。
- ステータス項目（カード数、今日やる数、未学習数、最終学習日）を、背景色付きの丸みを帯びたバッジ形式に変更。
- 各バッジにアイコンを追加し、配色をパステルカラー基調に調整。
- ステータス項目が占めていたグリッド領域を削除。

```typescript
// 期待される配色の例
// カード数: bg-blue-50 text-blue-600
// 今日やる数: bg-green-50 text-green-600
// 未学習数: bg-amber-50 text-amber-600
// 最終学習日: bg-purple-50 text-purple-600
```

## 検証計画

### 自動テスト
- 現状、UIコンポーネントのテストコードは存在しないため、ビルドが通ることを確認します。
- `npm run build` を実行してエラーがないか確認。

### 手動確認
1. ダッシュボードを表示し、フォルダ名と同じ行にステータス（アイコン・数値・ラベル）が表示されていることを確認。
2. デザインが以前よりもコンパクトで視覚的に洗練されていることを確認。
3. 画面幅を縮小した際に、ステータス項目が適切にレイアウトされるか確認。
