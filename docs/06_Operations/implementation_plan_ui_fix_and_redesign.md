# UI修正・トグルデザイン刷新計画

## 概要
1.  **カード編集画面**: 消えてしまった「耐性スコア」というラベルテキストを復元し、Indexロゴの右側に適切に配置します。
2.  **トグルスイッチ (Global)**: アプリ全体のスイッチ (`<Switch>`) デザインを一新し、よりコンパクトで可愛らしい見た目に変更します。

## 変更内容

### 1. `src/Components/card/CardEditor.tsx`
*   **ラベル復元**:
    *   編集により削除されてしまった `<span>耐性スコア</span>` を再配置します。
    *   **配置場所**: ヘッダーの `[Index Q番号]` の右隣、耐性スコアバッジのコンテナ内。
    *   **スタイル**: `text-[9px] md:text-[10px] font-bold text-slate-300 uppercase tracking-[0.3em]` を適用し、他のラベル（INDEX, TAGSなど）と統一感を持たせます。
    *   **レイアウト**: 親コンテナに `flex-col-reverse` (または `justify-end` + `flex-col`) を適用し、テキストがバッジの上にくるか、あるいはデザインに合わせて「Index」と同じベースラインに揃えます。
    *   ユーザー要望「Index Q◯の表示サイズと同じにして下さい」→ Indexブロックと同様に、上に小さいラベル、下にメイン要素（バッジ）が来るように構成します。

### 2. `src/Components/ui/switch.tsx`
*   **デザイン刷新**:
    *   **形状**: ボーダー (`border-2`) を削除し、フラットでクリーンな印象にします。
    *   **サイズ**: 全体を少し縮小し「コンパクト」にします。
        *   幅: `w-9` (36px) → `w-8` (32px)
        *   高さ: `h-5` (20px) → 維持、または視覚的に調整
    *   **Thumb (つまみ)**:
        *   サイズ: `h-4 w-4` → `h-3.5 w-3.5` (少し小さくし、余白を持たせる)
        *   影: `shadow-lg` → `shadow-sm` (柔らかい影)
    *   **アニメーション**: `input` の移動量を調整 (`translate-x-4` → `translate-x-3.5` 程度)。
    *   **色**: `data-[state=unchecked]:bg-slate-200` を維持しつつ、ボーダーが無い分、背景色で丸みを強調します。

## 検証計画
### 手動検証
1.  **カード編集画面**:
    *   任意のカードを開く。
    *   ヘッダーの「耐性スコア」バッジの上に、薄い文字で「耐性スコア」と表示されていることを確認する。
    *   その高さやレイアウトが左隣の「INDEX」と揃っていることを確認する。
2.  **トグルスイッチ**:
    *   **設定画面**を開く（多くのトグルがあるため）。
    *   トグルのデザインが変更されていることを確認する（枠線なし、少し小さめ）。
    *   クリックして ON/OFF がスムーズに切り替わるか確認する。
    *   **カード編集画面**の「下書き」スイッチなども同様に変更されているか確認する。
