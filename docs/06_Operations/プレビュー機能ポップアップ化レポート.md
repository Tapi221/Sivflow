# プレビュー機能ポップアップ化 実装レポート

## 概要
カード作成画面のプレビュー機能を、従来のインライン表示からダイアログ（ポップアップ）表示に変更しました。また、プレビュー内でカードを裏返して表示内容を確認できる機能を追加しました。

## 変更内容

### 1. プレビューのポップアップ化
- `src/Components/card/CardEditor.tsx` のプレビュー表示を `Dialog` コンポーネントを使用したポップアップ形式に変更しました。
- これにより、エディタのスクロール位置に関わらず、いつでも中央に完成イメージを表示できるようになりました。

### 2. ポップアップ内での裏返し機能
- プレビュー用ダイアログ内に表示される `Flashcard` コンポーネントに `onFlip` プロップを連携させました。
- プレビュー画面内のカードをクリック、または「答えを表示/問題を表示」ボタンを押すことで、実際の学習時と同じように表裏の切り替えが可能です。

### 3. UI調整
- ヘッダーのプレビュートグルを、アイコン付きの「プレビュー」ボタンに変更しました。
- ダイアログ背景に `backdrop-blur` を適用し、モダンでプレミアムな質感を演出しました。

## 確認事項
- [x] 「プレビュー」ボタン押下でダイアログが開く。
- [x] ダイアログ内でカードの表裏切り替えができる。
- [x] ダイアログを閉じても編集内容が保持されている。
- [x] 広い画面（1400px）でもダイアログが適切に中央表示される。
