# フォルダ一覧のボタン表示制御 (ホバー時表示) 実施報告

## 変更内容

フォルダ一覧をよりシンプルにし、重要な情報（名前やカード数）に集中できるように、「＋」と「︙」ボタンの表示制御を行いました。

### 変更点詳細

**`src/Components/folder/FolderTree.tsx`**

1.  **デフォルト非表示（透明化）**:
    - アクションボタンのコンテナに対して `opacity-0` を適用し、通常時は表示されないようにしました。
2.  **ホバー/タップ時の表示**:
    - フォルダ行（`group`）にマウスをホバー、またはタップした際に `opacity-100` となるよう設定しました。
3.  **スムーズなアニメーション**:
    - `transition-opacity duration-200` を追加し、ボタンが唐突に現れるのではなく、ふわっと浮かび上がるように調整しました。

## 検証結果

- **表示確認**:
  - 通常時はフォルダ名とメタデータのみの非常にクリーンなリストが表示されます。
  - 行の上にマウスを移動すると、スムーズに「＋」と「︙」ボタンが表示され、操作が可能になることを確認しました。
- **配置の安定性**:
  - `opacity-0` を使用しているため、ボタンが表示されてもレイアウトが左右にズレる（ガタつく）ことはありません。
