# 「無題のカード」表記削除の実装完了

## 概要
カードのタイトルやテキストが未設定の場合に表示されていた「無題のカード」というプレースホルダーテキストを削除しました。
これにより、意図せず「無題のカード」という文字が並ぶのを防ぎ、ユーザーが入力した情報のみが表示されるようになります。

## 実装内容

### 1. カード一覧 (`src/Components/card/CardList.tsx`)
*   タイトル表示ロジックを変更しました。
*   従来: タイトル → 質問テキスト → **「無題のカード」**
*   現在: タイトル → 質問テキスト → (表示なし)

### 2. ダッシュボード (`src/Pages/Dashboard.jsx`)
*   「作成中のドラフト (Drafts in Progress)」セクションでの表示ロジックを変更しました。
*   従来: タイトル → **「無題のカード」**
*   現在: タイトル → 質問テキスト → (表示なし)

## 検証方法
1.  **タイトル未設定のカード**:
    *   質問テキストが入力されている場合 → **質問テキスト**がタイトルとして表示されることを確認。
    *   質問テキストも未入力の場合 → タイトル部分が空欄（または高さのみ確保）となることを確認。
2.  **ドラフトカード**:
    *   ダッシュボードのドラフト一覧で、同様に「無題のカード」が表示されないことを確認。
