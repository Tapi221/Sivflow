# 表示モード（ツリー/カラム）の永続化実装計画

フォルダ管理画面において、ユーザーが選択した表示モード（ツリー表示またはカラム表示）を `localStorage` に保存し、ページをリロードしたり他の画面から戻ってきたりした際にも設定が維持されるようにします。

## 変更内容

### フォルダ管理画面

#### [MODIFY] [Folders.jsx](file:///c:/FlashcardMaster/src/Pages/Folders.jsx)
- **初期状態の取得**:
    - `viewMode` ステートの初期値を `useState('tree')` から、`localStorage` から取得する関数に変更します。
    - 保存されていない場合はデフォルトで `'tree'` を使用します。
- **状態変更時の保存**:
    - `viewMode` が変更された際に、新しい値を `localStorage` に書き込む副作用（`useEffect`）を追加します。

## 検証計画

### 手動確認
1. フォルダ管理画面で「カラム表示」に切り替える。
2. ブラウザをリロードし、カラム表示が維持されていることを確認する。
3. 他の画面（ダッシュボードなど）へ遷移してから戻り、カラム表示が維持されていることを確認する。
4. 再度「ツリー表示」に切り替え、同様にリロードして維持されることを確認する。
