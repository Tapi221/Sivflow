# カードグリッド表示のレイアウト崩れ修正計画

長い連続した文字列（英数字の羅列など）が含まれるカードがグリッド表示された際に、カードの幅がコンテンツに合わせて広がってしまい、レイアウトが崩れる問題を修正します。

## 修正内容

### [Component] [CardList.tsx](file:///c:/FlashcardMaster/src/Components/card/CardList.tsx)

#### [MODIFY] カードアイテム (`CardItem`) のスタイリング

- カードのルート `div` に `min-w-0` を追加し、Grid システム内での最小幅を 0 に設定します。これにより、コンテンツが幅を押し広げるのを防ぎます。
- タイトル (`h3`) および プレビューテキスト (`p`) に `break-words` および `overflow-wrap: anywhere` を追加し、長い文字列でも適切に折り返されるようにします。


## 検証計画

### 手動確認
1. 非常に長い英数字（スペースなし）をタイトルや本文に持つカードを作成する。
2. フォルダ画面（FolderView）の「CARDS」タブで、そのカードが他のカードと同じ幅で表示され、テキストが折り返されていることを確認する。
3. モバイル表示（レスポンシブ）でも横揺れが発生せず、適切に表示されることを確認する。
