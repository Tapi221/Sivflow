# 完了報告：ドラッグハンドルの表示切り替え（ホバー時のみ表示）

フォルダーツリーにおいて、並び替え用のドラッグハンドルが必要な時だけ表示されるように修正しました。

## 変更内容

### [FolderTree](file:///c:/FlashcardMaster/src/Components/folder/FolderTree.tsx)
- **表示制御の追加**: ドラッグハンドル（`GripVerticalIcon`）を包む `span` 要素に、以下のスタイルを追加しました。
    - 通常時: `opacity-0` （非表示）
    - ホバー時 (`group-hover`): `opacity-100` （表示）
    - クリック時 (`active`): `opacity-100` （表示）
    - ドラッグ中 (`snapshot.isDragging`): `opacity-100` （表示）
- **アニメーション**: `transition-opacity` を適用し、表示の切り替わりをスムーズにしました。
