# カード作成画面余白調整 実装レポート

## 概要
カード作成画面、特に一問一答モードにおいて、大画面使用時の左右の余白を縮小し、スペースを有効活用できるようにレイアウトを調整しました。

## 変更内容

### 1. 一問一答モードのコンテナ幅拡張
- `src/Pages/OneQAMode.jsx` において、コンテナの最大幅を `max-w-5xl` (1024px) から `max-w-[1400px]` に拡張しました。
- `px-4 md:px-8` を追加し、画面端との適切な距離を保ちつつ、中央のコンテンツエリアを広げました。

### 2. カードエディタの要素間隔調整
- `src/Components/card/CardEditor.tsx` において、問題セクションと解答セクションの間の `gap` を `gap-12` から `gap-8` に変更しました。
- コンテンツ左右のパディングを `md:px-4` から `md:px-6` に増やし、全体的なバランスを整えました。

## 確認事項
- [x] PCの広い画面で、カード作成エリアが従来より広く表示され、左右のデッドスペースが減少していること。
- [x] 1枚作成画面（`CardEdit.jsx`）と一問一答モード（`OneQAMode.jsx`）でコンテナ幅が 1400px に統一されていること。
- [x] レスポンシブ表示（タブレット・スマホ）でレイアウトが崩れていないこと。
