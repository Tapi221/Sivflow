# FolderView パンくずリスト位置調整の実装計画

## 目的
フォルダ表示画面（FolderView）において、パンくずリストを含むヘッダー行を画面上部ギリギリに配置する。

## 現状の課題
`src/Pages/FolderView.jsx` において、コンテナの上部パディングによりパンくずリストの位置が下がっている。

- コンテナ: `pt-2` (8px), `md:pt-4` (16px)

## 変更内容

### `src/Pages/FolderView.jsx` の修正

1.  **コンテナパディングの削減**
    - `pt-2` -> `pt-0` あるいは `pt-1` (モバイル)
    - `md:pt-4` -> `md:pt-1` (デスクトップ)

パンくずリストはヘッダー (`mb-2`) 内にあるため、このパディングを削減することで、コンテンツエリアの最上部に近づける。

## 検証計画
1.  ローカルでビルドし確認。
2.  パンくずリストの位置が期待通り画面上部に近づいているか確認。
