# 実装計画：フォルダー画面パンくずリストの左寄せ修正

前回の「右寄せ」の修正を撤回し、パンくずリストを左側（戻るボタンの隣）に配置するレイアウトに修正します。

## 課題
- パンくずリストを右側に配置したが、ユーザーは左側（特に戻るボタンの隣）を希望している。
- 戻るボタンが確実に左端にあり、その直後にパンくずリストが続くようにする。

## 変更内容

### [FolderView](file:///c:/FlashcardMaster/src/Pages/FolderView.jsx)
- **レイアウトの再構成**: 
    - ヘッダーを `[戻るボタン] [パンくずリスト (左詰め)] [操作ボタン (右詰め)]` の順に並べます。
    - パンくずリストのコンテナから `justify-end` を削除し、`flex-1` を適用して残りのスペースを左側から埋めるようにします。
    - マスクグラデーション（`mask-gradient-right`）を維持し、右側に溢れる場合にフェードアウトさせる。

## 検証計画
- [ ] 戻るボタンが左端にあることを確認。
- [ ] パンくずリストが戻るボタンのすぐ右隣から開始されていることを確認。
- [ ] 操作ボタン（選択モード、同期、メニュー等）が右端に維持されていることを確認。
- [ ] パンくずリストが長い場合に、アクションボタンを押し潰さずに適切に省略またはスクロールできることを確認。
