# UI改善（アクセントカラー・テキスト折り返し・タグ配置）検証手順

## 変更内容概要

1.  **アクセントカラー**: アプリ起動時（リロード時）に設定した色が即座に適用されるように修正（緑色の一瞬の表示を防止）。
2.  **テキスト折り返し**: カード内の日本語長文（句読点なし）などが枠外にはみ出す問題を `break-all` で修正。
3.  **配置変更**: 「Back to Question」ボタンと「タグ」をカードの枠外（右上）に移動。

## 検証手順

### 1. アクセントカラーの即時適用確認
1.  **設定** > **外観** からアクセントカラーをデフォルト（緑）以外の色（例: ピンク、青）に変更する。
2.  一度ブラウザをリロードする (F5 または Ctrl+R)。
3.  **期待値**: リロード直後、一瞬緑色になることなく、最初から設定した色（ピンク/青）で画面が表示されること。

### 2. テキスト折り返し確認
1.  **新規カード作成** で、以下のテキストを入力する。
    *   `ああああああああああああああああああああああああああああああああああああああああああああああああああああああああああああ` (60文字以上の連続した日本語)
2.  カードを表示（学習モードまたはプレビュー）。
3.  **期待値**: テキストがカードの右端で見切れず、適切に次の行に折り返されていること。

### 3. タグ・ナビゲーション配置確認
1.  タグ（例: `TestTag`）を付けたカードを用意する。
2.  そのカードの**学習モード**（またはプレビュー）を開く。
3.  カードを裏返して（クリックまたはスペースキー）、「Back to Question」ボタンを表示させる。
4.  **期待値**:
    *   「Back to Question」ボタンとタグが、白いカードの**枠外（上部）**に配置されていること。
    *   カード内のテキストや画像と重なっていないこと。
