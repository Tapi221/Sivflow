# 実装計画：ドラッグハンドルの表示制御（ホバー/操作時のみ表示）

フォルダーツリーのドラッグハンドル（並び替えボタン）が、必要な時だけ表示されるように視認性を調整します。

## 課題
- ドラッグハンドルが常に表示されているため、画面が少し煩雑に見える可能性がある。
- ユーザーは「ホバー時」と「クリック（操作）時」のみの表示を希望している。

## 変更内容

### [FolderTree](file:///c:/FlashcardMaster/src/Components/folder/FolderTree.tsx)
- ドラッグハンドルを包んでいる `span` タグのクラスを調整します。
- `opacity-0` を基本とし、以下の条件で `opacity-100` になるようにします。
    - `group-hover:opacity-100`: カード全体にマウスが乗った時。
    - `active:opacity-100`: クリック（タッチ）した時。
    - `snapshot?.isDragging` が `true` の時: ドラッグ中も表示を維持。
- スムーズな表示切り替えのために `transition-opacity` を追加します。

## 検証計画
- [ ] デスクトップでマウスをカードに乗せた際にドラッグハンドルが表示されることを確認。
- [ ] ドラッグハンドルをクリック/ドラッグしている間、表示が維持されることを確認。
- [ ] マウスを外した際にドラッグハンドルが非表示になることを確認。
- [ ] モバイル端末でタッチした際に表示されることを確認（ホバーがないため、タッチ/操作時のみの表示となる想定）。
