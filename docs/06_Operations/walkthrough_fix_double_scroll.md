# スクロール・レイアウト修正 完了報告

## 変更内容

### 1. 二重スクロールの解消
- サイドバー内部でスクロールバーが2重に表示される問題を修正しました。
- スクロール機能を親コンテナ（`TreeViewLayout`）に一本化しました。

### 2. 青い枠線（フォーカスリング）の削除
- サイドバーや編集エリアをスクロール/クリックした際に表示される青い枠線を非表示にしました (`outline-none`)。

### 3. 高さの最適化
- モバイルとデスクトップでヘッダー/フッターの高さが異なるため、それぞれに合わせた高さを計算するように変更しました。
  - **モバイル**: `100dvh - 152px` (ヘッダー + フッター + 余白)
  - **デスクトップ**: `100dvh - 40px` (ヘッダーなし、上部余白のみ)

## 確認方法
1. **スクロールバー**:
   - サイドバーにマウスを合わせてもスクロールバーが1本だけであることを確認してください。
2. **枠線**:
   - サイドバーや編集エリアをクリック・スクロールしても青い線が出ないことを確認してください。
3. **表示崩れ**:
   - 画面の上下に不自然な隙間がなく、コンテンツがコンテナ内に収まっていることを確認してください。
