# 設定画面 Glassmorphism 改修完了

## 概要
設定画面全体にGlassmorphism（すりガラス風）デザインを適用し、モダンで統一感のあるUIに更新しました。

## 変更内容

### 1. 共通スタイルの追加 (`src/index.css`)
以下のユーティリティクラスを追加しました：
- `.glass-sidebar`: サイドバー用の暗めのすりガラス背景
- `.glass-content`: メインコンテンツ用のすりガラス背景
- `.glass-card`: カード要素用の汎用すりガラスクラス
- `.glass-panel`: ダイアログ全体を包むパネル
- `.glass-button`: アクティブなタブボタン用のスタイル

### 2. 設定画面 (`SettingsDialog.jsx`)
- ダイアログ全体の背景を透明化し、Glassmorphismパネルを採用
- サイドバーとメインコンテンツを分離し、それぞれ異なる透過度とブラーを設定
- 各タブ（アカウント、学習設定、タグ、音声、ショートカット、同期、テーマ、データ）のデザインを更新
- モバイル表示時のヘッダーやレイアウトを調整

### 3. サブコンポーネントの更新
以下のコンポーネントにも同様のデザインを適用しました：
- `DataRescuePanel.tsx`: バックアップ・復元画面
- `BlockOrdering.tsx`: ブロック並び替え画面
- `DeviceSyncSettings.tsx`: デバイス同期管理画面

## Verification Steps
以下の手順でデザインを確認してください：

1. アプリケーションを起動し、設定画面を開く
2. 各タブを切り替えて、背景の透過やブラー効果が適用されていることを確認
3. ダークモードとライトモードの両方で視認性を確認
4. モバイルサイズでサイドバーが正しく表示されるか確認

## 補足
- `StorageManager` コンポーネントは現在使用されていないため、更新対象外としました。
