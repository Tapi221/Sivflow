# 実装計画：フォルダーアイテム背面背景の微調整

フォルダー管理画面において、白いフォルダーカードがより際立つように、その背面の背景色（コンテナ）を白から薄いグレー（微細なグレー）に調整します。

## 課題
- 現状、フォルダーアイテムとその背面の背景が共に白に近いため、ユーザーが追加した3D効果や影が背景に溶け込んでしまい、立体感が十分に発揮されていない。
- ユーザーは「フォルダの下（背面）」を薄いグレーにすることを希望している。

## 変更内容

### [FolderTree](file:///c:/FlashcardMaster/src/Components/folder/FolderTree.tsx)
- ルートおよびネストされた `Droppable` コンテナの背景色を `bg-slate-50/50` のような非常に薄いグレーに設定します。
- これにより、個々の白いフォルダーアイテム（`bg-white`）とのコントラストが生まれ、立体感が強調されます。

### [Folders](file:///c:/FlashcardMaster/src/Pages/Folders.jsx)
- メインのコンテンツカード（`bg-white`）内の、フォルダーツリーが表示されるエリアの背景を調整する、または全体の構成を微調整します。

### [FolderView](file:///c:/FlashcardMaster/src/Pages/FolderView.jsx)
- サブフォルダのチップ（丸いバッジ）が並ぶエリアの背景にも、同様に薄いグレーを適用し、統一感を出します。

## 検証計画
- [ ] フォルダ管理画面を開き、各フォルダカードの背面が薄いグレーになっていることを確認。
- [ ] フォルダカード自体は白のままで、影（3D効果）が背景に対してはっきりと見えることを確認。
- [ ] 子フォルダの階層（インデント部分）も一貫した背景色になっているか確認。
- [ ] フォルダ詳細画面（FolderView）のサブフォルダエリアも同様に調整されているか確認。
