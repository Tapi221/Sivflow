# タブUIデザイン改善計画

## 概要
`FolderView.jsx` 内のタブ（CARDS, MEMO, NOTES, MAP）のデザインが「チープ」であるとの指摘を受け、よりモダンで高品質なデザインへ改善します。

## 改善方針
1.  **アイコンの追加**: テキストだけでなくアイコンを添えることで、視認性とプロフェッショナル感を向上させます。
2.  **アニメーションの強化**: `framer-motion` を使用し、タブ切り替え時にインジケーターが滑らかにスライドするようにします。
3.  **タイポグラフィの調整**: フォントサイズ、ウェイト、レタースペースを微調整し、洗練された印象を与えます。
4.  **アクティブ状態の洗練**: 単なる下線ではなく、丸みのあるインジケーターや、アクティブ時の微細な背景変化を適用します。

## 変更内容

### [src/Pages/FolderView.jsx](file:///c:/FlashcardMaster/src/Pages/FolderView.jsx)

1.  **Lucideアイコンの追加インポート**
    - `Library`, `StickyNote` などを追加します（既存の `BookOpen`, `FileText`, `MapIcon` も使用）。

2.  **タブデザインの刷新**
    - `TabsTrigger` 内の構造を変更し、アイコンとラベルを配置します。
    - `framer-motion` の `layoutId` を使用して、アクティブなタブの下をスライドするインジケーターを実装します。
    - スタイルを `bg-transparent` ベースにし、ホバー時のインタラクションを追加します。

## 検証計画

### 手動検証手順
1.  **フォルダ詳細画面を開く**
2.  **タブの切り替え操作**
    - 「CARDS」「MEMO」「NOTES」「MAP」の各タブをクリック。
    - インジケーターがスムーズにスライドすることを確認。
    - アイコンとテキストの配置が崩れていないことを確認。
3.  **レスポンシブ確認**
    - 画面幅を縮小し、モバイル表示時でもタブが正しく並び、操作可能であることを確認。

## 補足
デザインは既存のアクセントカラー（primary-600）を基本としつつ、よりプレミアムな「OneQAモード」に近い質感を意識します。
