# カードエディタヘッダーレイアウト変更計画

## 概要
カード編集画面 (`CardEditor.tsx`) のヘッダーレイアウトを変更します。
「INDEX Q◯」ロゴの即右側に、ブックマークアイコンとはてなマーク（不確実フラグ）を縦並びで配置し、その右側にタイトル入力欄が続くように構成を変更します。

## 変更内容

### `src/Components/card/CardEditor.tsx`

1.  **ツールバーからの削除**:
    *   画面右側（PC表示時）または下部にあるツールバーから、ブックマークボタンと不確実フラグボタンを削除または非表示にします。
    *   残った要素（下書きスイッチなど）のレイアウトを調整します。

2.  **ロゴ横への配置**:
    *   「Index」ラベルと「Q番号」を表示している `div` (356行目付近) の構造を変更します。
    *   Q番号の右側に、新しいコンテナ (`flex flex-col`) を作成し、そこにブックマークボタン（上）と不確実フラグボタン（下）を配置します。
    *   **配置順序**: 上にブックマーク (`Bookmark`)、下にはてなマーク (`HelpCircle`)。

3.  **スタイル調整**:
    *   アイコンボタンは小さめのサイズ（縦並びで違和感がない程度）に調整します。
    *   タイトル入力欄との間隔が適切になるようにマージンやギャップを調整します。

## レイアウトイメージ
**変更前:**
```
[INDEX Q1]         [タイトル入力] [タグ]       [下書き|BM|Help]
(耐性スコア)
```

**変更後:**
```
[INDEX Q1] [BM]    [タイトル入力] [タグ]       [下書き]
           [Help]
(耐性スコア)
```

## 検証計画
### 手動検証
1.  **配置確認**:
    *   PCおよびモバイルビューで、INDEX Q番号の右側にアイコンが2つ縦に並んでいることを確認する。
    *   上がブックマーク、下がはてなマークであることを確認する。
    *   その右側にタイトル入力欄が配置されていることを確認する。
2.  **動作確認**:
    *   移動した各アイコンをクリックし、ブックマーク状態・不確実状態が正しく切り替わることを確認する。
