# UI改善：ブロック操作ボタンとカードヘッダーの調整

## 概要
ユーザーからのリクエストに基づき、以下の2点のUI改善を行います。
1.  **ブロック操作メニューの改善**: ボタンを小さく、円形に変更します。
2.  **カードヘッダーのレイアウト調整**: 「INDEX Q1/耐性スコア」（添付画像2）を「タイトル」入力欄と同じ高さ（水平ライン）に揃えて表示するようにします。

## 変更内容

### 1. ブロック操作ボタン (`src/Components/card/blocks/BlockWrapper.tsx`)
*   操作ボタン（複製、削除、ドラッグハンドル）のスタイルを変更します。
    *   `rounded-md` -> `rounded-full` (円形化)
    *   `p-1` -> `w-7 h-7 flex items-center justify-center p-0` (サイズ固定・縮小)
    *   アイコンサイズを微調整 (`w-3.5 h-3.5` -> `w-3 h-3`)

### 2. カードヘッダーレイアウト (`src/Components/card/CardEditor.tsx`)
*   カード上部のIndex/Score表示部分と、タイトル入力欄のコンテナ構造を見直します。
*   現在は `flex-col md:flex-row` でラップされていますが、配置や `items-center` の適用を確認し、視覚的に「同じ高さ」に見えるように調整します。
*   具体的には、Index/Scoreセクションのパディングやマージンを調整し、タイトル入力ボックスの中心線と揃うようにします。

## 検証計画
### 手動検証
1.  **ブロック操作**: ホバー時に表示されるメニューが円形かつ小さくなっていることを確認。
2.  **ヘッダー配置**: PC画面幅において、左側の「INDEX Q1」セクションと右側の「タイトル」入力欄が垂直方向の中央で揃っている（または上端が揃っているなど、違和感のない配置）ことを確認。
