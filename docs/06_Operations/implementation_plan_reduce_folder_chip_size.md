# フォルダ画面のチップUIサイズ縮小

フォルダ画面（特にFolderView内）のサブフォルダ・新規フォルダボタンのサイズを、ユーザーの要望に合わせて大幅に縮小します。

## 概要
`src/Pages/FolderView.jsx` 内のサブフォルダ一覧を表示しているチップ（ピル型ボタン）のパディング、フォントサイズ、アイコンサイズを縮小し、よりコンパクトな表示にします。

## 変更内容

### UI調整
#### [MODIFY] [FolderView.jsx](file:///c:/FlashcardMaster/src/Pages/FolderView.jsx)
- **新規フォルダボタン（button）**
    - パディング: `px-4 py-2.5` → `px-2.5 py-1.5`
    - アイコンサイズ: `w-4 h-4` → `w-3 h-3`
- **サブフォルダチップ（div）**
    - パディング: `px-4 py-2.5` → `px-2.5 py-1.5`
    - フォントサイズ: `text-xs` → `text-[10px]`
    - アイコン（FolderOpen）サイズ: `w-4 h-4` → `w-3 h-3`
    - カード数バッジ: `text-[10px] px-1.5 py-0.5` → `text-[9px] px-1 py-0`
- **全体のギャップ**: `gap-3` → `gap-2`

## 検証プラン

### 手動確認
1. アプリを起動し、フォルダ詳細画面 (`FolderView`) に遷移する。
2. 上部のサブフォルダチップと「新規フォルダ」ボタンのサイズが縮小されていることを確認する。
3. ラベルが切れていないか、クリック・タップしやすいかをモバイル表示も含めて確認する。
