# カード内タイトルの非表示化 実装計画

## 現状の課題
カード表示画面（`Flashcard.tsx`）において、カードのコンテンツエリア上部にカードのタイトルが表示されています。
しかし、ページ上部のヘッダー部分にも既にタイトルが表示されているため、重複しており、カード内のスペースを無駄に消費しています。

## 修正内容
`Flashcard.tsx` の質問表示部分において、`cardData.title` を描画しているブロックを削除します。

### [Component] Card
#### [MODIFY] [Flashcard.tsx](file:///c:/FlashcardMaster/src/Components/card/Flashcard.tsx)
- 質問表示エリア内のタイトル描画ロジック（`{cardData.title && ...}`）を削除。
- 合わせて、前回追加したが不要となったヒント関連の変数（`hasHint`, `hint`, `showHint`）や `Zap` アイコンのインポート、および未使用のステートをクリーンアップします。

## 完了条件
- カードを表示した際、カード内部にタイトルが表示されないこと。
- ヘッダー部分のタイトル表示は維持されていること。

## 検証計画
1. アプリを起動し、任意のカードを表示する。
2. カード内部にタイトル（およびタイトル下の装飾ライン）が表示されていないことを確認する。
3. カードを裏返し、回答表示が崩れていないことを確認する。
4. 本番環境にデプロイし、実際の端末で確認する。
