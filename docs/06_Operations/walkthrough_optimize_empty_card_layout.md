# 空のカード編集時のレイアウト最適化 実施報告

## 変更内容

### 1. CardShellのレイアウト調整 (`src/index.css`)
- `.card-shell--resizable` の `min-height` を `calc(var(--card-w) * 3 / 4)` (約480px) から `144px` に変更しました。
- これにより、ブロックが少ない状態でもカードの高さが強制的に大きくならず、コンテンツに合わせて縮小されるようになりました。

### 2. BlockEditorのプレースホルダー圧縮 (`src/Components/card/BlockEditor.tsx`)
- 「ブロックを追加してください」の表示エリアのパディングを `py-12` から `py-6` に半減させました。
- 空の状態での高さを抑え、見た目をコンパクトに改善しました。

## 確認事項

- [x] **空の状態**: 新規カード作成時などに、無駄な余白がなくコンパクトに表示されること。
- [x] **ブロック追加**: ブロックを追加すると、それに合わせてカードの高さが自然に伸びること。
- [x] **リサイズ機能**: カード下部のハンドルを使って手動で高さを変更できる機能が維持されていること。
