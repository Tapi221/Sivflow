# 外観モード削除・ブロック並び替え制限・耐性スコア移動計画

## 概要
ユーザーの要望に基づき、UIの調整を行います。
1.  **設定画面**: 「外観モード（テーマ）」設定を削除します。
2.  **ブロックエディタ設定**: 並び替え時の移動方向を上下（Y軸）のみに制限します。
3.  **カード編集画面**: ヘッダーのレイアウトを変更し、耐性スコアの表示位置を「ブックマーク＆はてなマーク」と「タイトル」の間に移動します。

## 変更内容

### 1. `src/Components/settings/SettingsDialog.jsx`
*   **サイドバー項目削除**:
    *   `sidebarItems` 配列から `id: 'theme'` の項目を削除します。
    *   これにより、サイドバーから「テーマ」が消え、設定画面にアクセスできなくなります。
    *   (既存のテーマ設定自体は維持されますが、UIからは見えなくなります)

### 2. `src/Components/settings/BlockOrdering.tsx`
*   **ドラッグ制限**:
    *   `<Draggable>` コンポーネントに `lockAxis="y"` プロパティを追加します。
    *   これにより、設定画面でのブロック並び替えが上下方向のみに固定されます。

### 3. `src/Components/card/CardEditor.tsx`
*   **レイアウト変更**:
    *   現在のヘッダー左側コンテナ (`Index Q番号`などを含む部分) の構造を変更します。
    *   現在は「ロゴ＆アイコン」の行と「耐性スコア」の行が縦積み (`flex-col`) になっていますが、これを解消します。
    *   **変更後の並び**: `[Index Q番号] [アイコン(縦)] [耐性スコア] [タイトル...]` が横一列（PC表示時）になるように調整します。
    *   具体的には、`md:flex-col` クラスを削除し、必要に応じてコンテナを統合または `flex-row` に変更します。

## レイアウトイメージ (カード編集画面・PC)
**変更前:**
```
[Index Q1] [BM/Help]
(耐性スコア)
```

**変更後:**
```
[Index Q1] [BM/Help]  (耐性スコア)  [タイトル入力欄...]
```

## 検証計画
### 手動検証
1.  **設定画面**:
    *   設定ダイアログを開き、サイドバーに「テーマ」が存在しないことを確認する。
    *   他のタブ（アカウント、学習設定など）が正常に表示されることを確認する。
2.  **ブロック並び替え**:
    *   学習設定 > ブロックエディタ設定を開く。
    *   ブロックをドラッグし、左右に動かしても横へズレない（上下のみ移動する）ことを確認する。
3.  **カード編集画面**:
    *   カード編集画面を開く。
    *   ヘッダー左上が横並びになり、耐性スコアがアイコンの右側に配置されていることを確認する。
    *   モバイル表示でも著しく崩れていないか確認する（モバイルはスペースの都合上、適宜折り返しなどを許容するが、基本構造は維持）。
