# 完了レポート：カレンダー画面のボタンレイアウト修正

カレンダー画面の「Today」ボタンおよび月表示コントロールが同期ボタンと重なる問題を修正しました。

## 実施内容

### [Calendar.jsx](file:///c:/FlashcardMaster/src/Pages/Calendar.jsx)
- ヘッダー部分の `justify-between` を `justify-start` に変更し、コントロール群を左側に寄せました。
- レイアウト調整用の空の `div` を削除しました。

## 修正後のレイアウト（イメージ）
- **左側**: 「Today」ボタン、月表示（前月・次月移動ボタン）
- **右側**: 同期状態インジケーター（Layout.tsxにより固定配置）

これにより、画面右上の同期ボタンとカレンダーの操作部が重なることがなくなりました。

## 検証結果
- デスクトップ版：ボタンが左寄せになり、同期ボタンとの重複が解消された。
- モバイル版：既存のスタックレイアウト（`flex-col`）が維持され、操作性に問題がないことを確認。
