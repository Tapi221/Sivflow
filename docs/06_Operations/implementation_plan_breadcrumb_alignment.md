# 実装計画：フォルダー画面パンくずリストの右寄せ

フォルダー画面のヘッダーにあるパンくずリストを左側から右側（操作ボタンの左）に移動し、レイアウトを調整します。

## 変更内容

### [FolderView](file:///c:/FlashcardMaster/src/Pages/FolderView.jsx)
- **レイアウト構造の変更**: 
    - 現在パンくずリストが含まれている左側の `flex items-center gap-4` からパンくずリストを切り出し、右側の操作ボタンエリアの直前に配置します。
    - パンくずリスト自体に `flex-1` または `justify-end` を適用し、右側に寄るように調整します。
- **モバイル対応**: パンくずリストが長くなった場合に他のボタンを押し出さないよう、適切に `min-w-0` や `truncate` を維持します。

## 検証計画
- [ ] パンくずリストが画面の右側（アクションボタンの左隣）に表示されていることを確認。
- [ ] フォルダー階層が深くパンくずリストが長い場合に、レイアウトが崩れず適切に省略（truncate）されることを確認。
- [ ] モバイル表示でパンくずリストとボタンが適切に配置されていることを確認。
