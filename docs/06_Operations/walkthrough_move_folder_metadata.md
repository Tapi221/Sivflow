# フォルダ管理画面：メタデータのインライン表示 実施報告

## 変更内容

フォルダの「カード枚数」と「最終学習日」の情報を、タイトルの下に配置する従来のデザインから、タイトルの右側にインライン表示するデザインに変更しました。これにより、1つのフォルダが表示する縦幅がさらに縮小され、一覧性が向上しました。

### 変更点詳細

**`src/Components/folder/FolderTree.tsx`**

- **レイアウト変更**:
  - タイトルとメタデータを `flex-row` (横並び) に変更しました。
  - `flex-wrap` を適用し、画面幅が狭い場合でもレイアウトが崩れにくいようにしました。
- **スタイル調整**:
  - カード枚数と最終学習日のフォントサイズを `text-[10px] md:text-[11px]` に統一しました。
  - 透明度 (`opacity`) を調整し、タイトルとの視覚的な階層を明確にしました。

## 検証結果

- **表示確認**:
  - メタデータがタイトルの横に表示され、行の高さが抑えられていることを確認しました。
  - 「未学習」や「今日」「○日前」などの表示が正しく行われていることを確認しました。
