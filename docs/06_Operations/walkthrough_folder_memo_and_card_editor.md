# フォルダメモ複数対応とカード編集機能強化の実装完了

## 実装概要

ユーザー要望に基づき、以下の機能を実装しました。

1.  **フォルダメモの複数対応**
    - フォルダ詳細画面で複数のメモを追加・管理できるようになりました。
    - 各メモに対して画像の添付、編集、削除が可能です。
    - 従来のメモデータは自動的に最初の一つのメモとして表示されます（マイグレーション）。

2.  **カード編集画面の機能強化**
    - `Index Q〇` の左側に操作ボタンを追加しました。
    - **削除ボタン**: 現在編集中のカードを削除できます（確認ダイアログあり）。
    - **並び替えボタン**: 前後のカードと順序を入れ替えることができます。

## 変更されたファイル

- `src/types/index.ts`: `Folder` 型に `memos` フィールドを追加。
- `src/Components/folder/FolderMemo.tsx`: 複数のメモをリスト表示し、CRUD操作を行うUIへ刷新。
- `src/Pages/CardEdit.jsx`: 削除と並び替えのロジック(`deleteCard`, `updateCard`)を実装し、`CardEditor` へ渡す。
- `src/Components/card/CardEditor.tsx`: 新しいProps (`onDelete`, `onMoveUp`, `onMoveDown`) を受け取り、ヘッダー部分にボタンを表示。

## 検証手順

### 1. フォルダメモ機能
1. 任意のフォルダを開き、「MEMO」タブを選択します。
2. **新規追加**: 「メモを追加」ボタンをクリックし、テキストと画像を入力して保存します。
3. **リスト表示**: 保存したメモがリストに追加されることを確認します。
4. **編集**: 既存メモの編集ボタン（鉛筆アイコン）を押し、内容を変更して保存します。
5. **削除**: 削除ボタン（ゴミ箱アイコン）を押し、メモが消えることを確認します。

### 2. カード編集機能
1. カード編集画面（または新規作成画面）を開きます。
2. 左上の `Index Q〇` 横のアイコンを確認します。
    - 新規作成時やカードが1枚しかない場合は並び替えボタンは非活性または非表示です。
3. **並び替え**:
    - 複数のカードがある状態で、上矢印・下矢印ボタン（横向き矢印の回転）をクリックし、カードの順序が入れ替わることを確認します。
    - インデックス番号（Q番号）が即座に変わるか、一覧に戻って順序が変わっていることで確認します。
4. **削除**:
    - ゴミ箱ボタンをクリックし、確認ダイアログで「OK」を選択するとカードが削除され、フォルダ画面に戻ることを確認します。
