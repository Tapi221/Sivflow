# タグによる絞り込み機能の実装完了

## 概要
カード一覧画面 (`FolderView`) に、タグを選択して表示するカードを絞り込む機能（タグフィルター）を実装しました。
これにより、大量のカードの中から特定のテーマやカテゴリ（タグ）に属するカードだけを素早く見つけることが可能になります。

## 実装内容
`src/Pages/FolderView.jsx` に以下の変更を加えました。

1.  **タグデータの取得**:
    *   `useTags` フックを使用し、現在のフォルダ階層（ルートフォルダ）に関連するタグ一覧を取得します。
    *   パンくずリストからルートフォルダを自動的に判定するロジックを追加しました。

2.  **フィルターUIの追加**:
    *   画面右上のツールバー（表示切り替えボタンの左）に「フィルターアイコン」を追加しました。
    *   クリックすると、利用可能なタグの一覧がドロップダウンメニューとして表示されます。
    *   各タグにはチェックボックスが付いており、複数のタグを選択可能です。
    *   タグを選択すると、アイコン横に選択中のタグ数がバッジ表示されます。

3.  **絞り込みロジック**:
    *   **OR条件**でフィルタリングします。つまり、選択したタグの**いずれか一つでも**持っているカードが表示されます。
    *   タグが選択されていない場合は、通常通り全てのカードが表示されます。
    *   「全選択解除」ボタンで、素早くフィルターをリセットできます。

## 検証方法
1.  **事前準備**:
    *   いくつかのカードに異なるタグ（例: "重要", "復習", "文法"）を付与してください。
2.  **フィルタリングの確認**:
    *   フォルダ画面を開き、右上のフィルターアイコンをクリックします。
    *   タグリストが表示されることを確認します。
    *   "重要" タグを選択し、"重要" タグが付いたカードのみが表示されることを確認します。
    *   さらに "復習" タグを選択し、"重要" **または** "復習" タグが付いたカードが表示されることを確認します。
3.  **解除の確認**:
    *   チェックボックスを外すと、フィルターが解除されることを確認します。
    *   「全選択解除」ボタンですべての選択が外れ、全カードが表示されることを確認します。
