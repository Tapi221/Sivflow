# サブフォルダ一覧の余白縮小

## 概要
ユーザーからの要望「サブフォルダで並列関係のフォルダどうしはフォルダ間の余白をめっちゃ狭くしてください」に対応するため、`FolderTree.tsx` におけるサブフォルダ（レベル1以上）のアイテム間のマージンとギャップを大幅に縮小します。

## 変更内容

### Components/folder

#### [MODIFY] [FolderTree.tsx](file:///c:/FlashcardMaster/src/Components/folder/FolderTree.tsx)
- [x] **FolderItem コンポーネント (ルート div)**:
  - `mb-2` (8px) を、`level` が 0 より大きい場合は `mb-0.5` に変更しました。
- [x] **サブフォルダ展開時のコンテナ (render children part)**:
  - `gap-2` (8px) を `gap-0.5` (2px) に変更しました。
- [x] **FolderItem コンポーネント (インナーカード div)**:
  - サブフォルダの場合はパディングを `p-1.5 md:p-2` に縮小しました。

## 検証計画
- ローカルサーバーで確認。
- サブフォルダを作成し、複数並べた際の余白が「めっちゃ狭く」なっているか確認。
- ルートフォルダの余白は以前のまま（あるいは適切な広さ）であることを確認。
- ドラッグ&ドロップ時の挙動（ドロップ領域）に影響がないか確認。
