# フォルダ削除の確認警告の削除

## 目的
ユーザーの利便性向上のため、フォルダ削除時の確認ダイアログを廃止し、削除ボタンのクリックで即座にごみ箱へ移動（論理削除）されるようにします。ごみ箱から復元可能なため、即時削除によるリスクは許容されます。

## 提案される変更点

### フォルダページ

#### [MODIFY] [Folders.jsx](file:///c:/FlashcardMaster/src/Pages/Folders.jsx)
- **handleDeleteFolder の変更**:
    - ダイアログを表示する代わりに、直接 `handleConfirmDelete(folder)` を呼び出すように変更します。
- **不要なコードの削除**:
    - `DeleteFolderDialog` のインポートとコンポーネント配置を削除します。
    - `deleteDialogOpen`, `deletingFolder` などの関連ステート、および `getDescendantStats` 関数を削除します。

## 検証計画

### 手動確認
1. フォルダのメニューから「削除」を選択する。
2. 警告ダイアログが表示されず、即座にフォルダがリストから消える（ごみ箱へ移動する）ことを確認。
3. 「ごみ箱」画面で、削除したフォルダが正しく存在することを確認。
