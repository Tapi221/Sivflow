# 設定画面のアクセントカラー強化実装計画

設定画面のデザインを、ユーザーが選択したアクセントカラーをより効果的に取り入れた「プレミアム」な外観に改善します。

## 変更内容の概要

1.  **サイドバーの改善**: 単なるグレー背景から、アクセントカラーの非常に淡い色調に変更し、アクティブなタブをより際立たせます。
2.  **セクションヘッダーの強調**: 各設定セクションのタイトルに、アクセントカラーの装飾（左側のバーや下線）を追加します。
3.  **コンテナ背景の調整**: 各設定項目の背景（現在は `#F8FAFB`）を、アクセントカラーに基づいた微細なトーンに変更します。
4.  **インタラクティブ要素の統一**: スイッチ、スライダー、セレクトボックスなどのフォーカス状態やアクティブ状態をアクセントカラーで統一します。

## 提案される変更

### 設定コンポーネント

#### [MODIFY] [SettingsDialog.jsx](file:///c:/FlashcardMaster/src/Components/settings/SettingsDialog.jsx)

-   **サイドバー**:
    -   背景を `bg-[#F8FAFB]` から `bg-primary-50/30` に変更。
    -   アクティブなタブに `border-l-4 border-primary-600` を追加し、視認性を向上（デスクトップ）。モバイルでは下線を強調。
-   **セクションタイトル**:
    -   `h2` タイトルの横にアクセントカラーの垂直バー（`w-1.5 h-6 bg-primary-600 rounded-full`）を追加。
-   **セクション内項目**:
    -   背景色を `bg-[#F8FAFB]` から項目の重要度に応じて `bg-primary-50/40` または `bg-slate-50` に使い分け。
    -   ボーダーカラーを `border-slate-100` から `border-primary-100/50` などに調整。
-   **テーマ設定（カラーセレクター）**:
    -   選択中のカラーをより際立たせる 애니ーション（微かな拡大など）を追加。

## 確認計画

### 手動確認
1.  **アクセントカラー変更の反映**:
    -   「テーマ」設定からアクセントカラーを変更し、設定画面全体の配色（サイドバー背景、ヘッダー装飾、アイコン色）が即座に同期して変化することを確認。
2.  **モバイルレイアウトの確認**:
    -   モバイル表示時に、上部のタブバーの背景やアクティブ状態の表示が崩れていないか確認。
3.  **ダークモード/ライトモードの調和**:
    -   各テーマモードでアクセントカラーが背景と適切なコントラストをポテンシャルな視認性を含めて確認。

### 自動検証
-   ビルド確認: `npm run build` でエラーが出ないことを確認。
