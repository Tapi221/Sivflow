# 完了報告：フォルダー画面パンくずリストの左寄せ修正

パンくずリストを左側（戻るボタンの隣）に再配置し、レイアウトを最適化しました。

## 変更内容

### [FolderView](file:///c:/FlashcardMaster/src/Pages/FolderView.jsx)
- **レイアウトの再構成**: 
    - 戻るボタンを左端に固定し、そのすぐ右隣からパンくずリストが開始されるようにしました。
    - パンくずリストのコンテナから右寄せ設定（`justify-end`）を削除し、左側から順に並ぶように修正しました。
- **視認性の向上**: 
    - パンくずリストに `flex-1` を適用し、中央から右側にかけての空きスペースを埋めるようにしました。これにより、アクションボタン（右端）との明確な分離が維持されます。
- **レスポンシブ維持**: 
    - パンくずリストが長い場合に右側に溢れるエフェクト（マスクグラデーション）を維持し、横スクロールによる閲覧を可能にしています。
