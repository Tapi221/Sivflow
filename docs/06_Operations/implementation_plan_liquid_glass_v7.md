# 実装計画：Liquid Glass V7 統一パネル化

「行ごとのガラス帯」を廃止し、リスト全体を「1枚の大きなガラスパネル」として表現することで、iOS風の洗練されたUIを実現します。

## ユーザーレビューが必要な項目
> [!IMPORTANT]
> - フォルダ一覧全体を包む大きなガラスパネル（`liquid-panel`）を導入します。
> - 各フォルダ行は通常時「完全に透明」になり、リスト全体が1枚のガラス板の上に載っているように見えます。
> - 行の高さが約10-15%コンパクトになり、一覧性が向上します。

## 変更内容

### 1. スタイル定義の刷新 (`index.css`)
- **`.liquid-panel` の新設**:
  - 重厚なブラー（20px前後）と深い影（0 30px 80px）を適用。
  - 内側に 1px の白いヘアライン（ストローク）を入れ、エッジの質感を強調。
- **`.liquid-glass-row` のフラット化**:
  - `background: transparent`, `backdrop-filter: none`, `box-shadow: none` にリセット。
  - ホバー時および選択時のみ「薄い光の膜」のようなハイライト（`rgba(255,255,255,0.08)`）を適用。
- **タイポグラフィの最適化**:
  - フォルダ名を `rgba(255,255,255,0.95)`, `font-weight: 600` に、サブ情報を `rgba(255,255,255,0.6)` に調整。

### 2. コンテナ構造の変更 (`Folders.jsx`)
- `FolderTree` を包むフラグメントを `<div className="liquid-panel p-2 md:p-6 mb-20 md:mb-0">` に置き換え。

### 3. 行レイアウトの微調整 (`FolderTree.tsx`)
- 行のパディングおよびマージンを削減し、よりタイトでモダンな垂直リズムを作成。

## 検証計画

### 自動テスト
- `npm run build` を実行し、ビルドエラーがないことを確認。

### 手動検証（ブラウザ）
1. **全体構造の確認**:
   - フォルダリスト全体が1枚のガラスパネルとして浮き上がって見えるか。
   - 画面の背景がパネル越しに美しく透けているか。
2. **インタラクションの確認**:
   - 行をホバーした際、過度なガラス効果ではなく、上品なハイライトが入るか。
   - 選択中の行が埋もれず、かつリスト全体の透明感を損なっていないか。
3. **視認性の確認**:
   - 白いテキストがガラスパネルの上で十分なコントラストを持っているか。
