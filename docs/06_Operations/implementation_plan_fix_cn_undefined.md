# 実装計画：cn undefinedエラーの修正

## 概要
デプロイ環境で発生している `ReferenceError: cn is not defined` を修正します。原因は `CardEditor.tsx` で行っていた `cn` 関数のローカル定義が、プロダクションビルド（ミニファイ時）において、依存している他のコンポーネントやユーティリティとの間でスコープの不整合を引き起こしているためと考えられます。

## 変更内容

### [カードエディタコンポーネント]

#### [修正] [CardEditor.tsx](file:///c:/FlashcardMaster/src/Components/card/CardEditor.tsx)
- `cn` 関数のローカル定義（42-44行目付近）を削除します。
- ローカル定義のために使用していた `clsx` と `tailwind-merge` のインポートを削除します。
- 38行目付近でコメントアウトされていた `import { cn } from '@/lib/utils';` を有効化します。
