フォルダ管理画面のフォルダリストアイテムの縦幅を縮小し、一覧性を向上させました。

### 変更点詳細

**`src/Components/folder/FolderTree.tsx`**

- **パディングの縮小**:
  - `p-2.5 md:p-3.5` → `p-2 md:p-2.5` に変更しました。
- **マージンの縮小**:
  - `mb-2` → `mb-1.5` に変更しました。
- **アイコンコンテナサイズの縮小**:
  - `w-9 h-9 md:w-10 md:h-10` → `w-8 h-8 md:w-9 md:h-9` に変更しました。
- **レイアウト調整**:
  - 各要素間のギャップ (`gap`) を `gap-2.5` → `gap-2`、`gap-4` → `gap-3` など微調整し、全体のバランスを整えました。
