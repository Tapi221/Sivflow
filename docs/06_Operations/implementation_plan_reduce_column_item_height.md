# カラム表示のフォルダ縦幅削減

カラム表示（Finderスタイル）において、フォルダとカードの縦幅が広いため、一度に表示できる項目数が限られています。これを改善するために、余白（パディング）を調整して、よりコンパクトな表示にします。

## 変更内容

### [Components]

#### [MODIFY] [FolderColumn.tsx](file:///c:/FlashcardMaster/src/Components/folder/FolderColumn.tsx)

- **ヘッダーの調整**:
    - `py-2` (8px) を `py-1` (4px) に変更し、ヘッダーの高さを抑えます。
- **フォルダアイテムの調整**:
    - コンテナの垂直パディング `py-2` (8px) を `py-1` (4px) に削減します。
    - アイコンコンテナのパディング `p-1.5` (6px) を `p-1` (4px) に削減します。
    - フォルダ名のフォントサイズを `text-[13px]` から `text-[12px]` に微調整し、よりコンパクトにします。
- **カードアイテムの調整**:
    - カードアイテムの垂直パディング `py-1.5` (6px) を `py-1` (4px) に削減します。
- **作成用入力フィールドの調整**:
    - `py-2` を `py-1` に削減します。

## 検証計画

### 手動確認
- カラム表示に切り替え、フォルダおよびカードが以前よりコンパクトに表示されていることを確認する。
- ホバー時のアクションボタン（+、...）が正しく表示され、クリック可能であることを確認する。
- インライン作成時の入力フィールドの高さがアイテムと一致していることを確認する。
