# フォルダ一覧の縦幅をさらに縮小

## 概要
ユーザーからの要望「フォルダの縦幅をもう少し縮小して」に対応するため、アイコンサイズ、パディング、マージンをさらに削り、一覧性を極限まで高めます。

## 変更内容

### Components/folder

#### [MODIFY] [FolderTree.tsx](file:///c:/FlashcardMaster/src/Components/folder/FolderTree.tsx)

- [x] **アイコンコンテナサイズの縮小**:
  - アイコンコンテナを `w-7 h-7 md:w-8 md:h-8` に、アイコン自体を `w-3.5 h-3.5 md:w-4 h-4` に縮小しました。

- [x] **パディングのさらなる縮小**:
  - ルートフォルダ (`level === 0`): `p-1.5 md:p-2` に変更しました。
  - サブフォルダ (`level > 0`): `p-1 md:p-1.5` に変更しました。

- [x] **外側マージンの調整**:
  - ルートフォルダ: `mb-1.5`
  - サブフォルダ: `mb-0`

- [x] **右側の操作ボタン (Plus, MoreVertical)**:
  - ボタンサイズを `h-7 w-7` に、Plusアイコンを `w-4 h-4` に縮小しました。
