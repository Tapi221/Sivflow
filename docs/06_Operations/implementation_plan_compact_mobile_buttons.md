# カード編集画面のブロックボタン最適化（モバイル）

モバイル表示において、ブロック追加ボタンをアイコンのみにし、よりコンパクトなレイアウトに変更します。

## 修正内容

### [Component] [BlockEditor.tsx](file:///c:/FlashcardMaster/src/Components/card/BlockEditor.tsx)

#### [MODIFY] ブロック追加ボタンのスタイリング

- ボタン内のテキストラベル (`span`) に `hidden md:inline` を追加し、モバイルでは非表示にします。
- ボタン本体のクラスを調整し、モバイルでは `px-0 w-8` (アイコンのみのサイズ) とし、PCでは `px-4 w-auto` (テキストありのサイズ) になるようにします。
- `flex items-center gap-...` を `flex items-center justify-center gap-...` に変更し、アイコンのみの際に中央配置されるようにします。

## 検証計画

### 手動確認

1. モバイル端末（または開発者ツールのモバイルエミュレーション）でカード編集画面を開く。
2. ブロック追加ボタン（テキスト、コード、画像、音声、リンク、数式）がアイコンのみの円形ボタンになっており、1行にコンパクトに収まっていることを確認する。
3. PC表示では引き続きテキストラベルが表示されていることを確認する。
