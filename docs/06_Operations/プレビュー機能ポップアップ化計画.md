# プレビュー機能のポップアップ化計画 (Flip機能追加)

## 概要
カード作成画面のプレビュー機能をダイアログ（ポップアップ）表示に変更し、さらにダイアログ内でカードを裏返して（表面・裏面）確認できる機能を追加します。

## Proposed Changes

### カードエディタ (CardEditor)
#### [MODIFY] [CardEditor.tsx](file:///c:/FlashcardMaster/src/Components/card/CardEditor.tsx)
- `Dialog` 関連コンポーネントをインポートします。
- プレビューの表裏状態を管理する `isPreviewFlipped` ステートを追加します。
- ヘッダーのトグルを「プレビュー」ボタンに変更します。
- `Dialog` 内に `Flashcard` をレイアウトし、`onFlip` プロップを渡して表裏を切り替えられるようにします。
- インラインのプレビューセクションを削除します。

## Verification Plan

### 手動確認
1. 一問一答モードで「プレビュー」ボタンをクリック。
2. ポップアップでカードが表示される。
3. ポップアップ内の「答えを表示/問題を表示」ボタン、またはカードクリックで表裏が切り替わることを確認。
4. 画像やコード、音声ブロックなどの各要素がプレビュー内で正しく表示・動作することを確認。

### 自動ビルド確認
- `npm run build` を実行。
