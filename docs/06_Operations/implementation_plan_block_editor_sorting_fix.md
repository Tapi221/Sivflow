# 設定画面のブロックエディタ並び替え修正計画

## 現状の課題

- 設定画面の「ブロックエディタ設定」において、項目のドラッグ＆ドロップによる並び替えが期待通りに動作していない。
- ユーザー提供の画像では、ドラッグ中のアイテムがレイアウトから浮いているが、並び順が正しく更新されない、または表示が崩れる可能性がある。

## 調査対象

- `src/Components/settings/SettingsDialog.jsx`: ブロックエディタ設定のUI実装
- `src/hooks/useSettings.ts` (または関連する設定フック): 設定の保存ロジック
- 使用されているドラッグ＆ドロップライブラリ (例: `@hello-pangea/dnd`, `dnd-kit` など) の実装状況

## 修正方針

1. 並び替えロジック（`onDragEnd` など）が正しく実装されているか確認。
2. 並び替え後の状態が適切に永続化（保存）されているか確認。
3. ドラッグ中のプレースホルダーやアニメーションが正しく動作するようにCSSまたはライブラリのプロパティを調整。

## 検証計画

- ローカル環境でドラッグ＆ドロップを試行し、挙動を再現。
- 並び替え後に設定を保存し、再読み込み後も順序が維持されているか確認。
