# 音声マークとリンクマークの配置・サイズ統一の実装計画

カード画面における視覚的整合性を高めるため、音声再生マークをリンクマーク（参考リンクバッジ）の左側に配置し、デザイン（サイズ・形状）を統一します。

## 変更内容

### 1. Flashcard.tsx (カード表示・プレビュー)
- [MODIFY] `actionsBottomRight` のロジックを変更。
  - カードに音声が存在する場合、音声再生用のバッジを**最初に追加**します。
  - リンクバッジ（参考リンク）がある場合、その**後に追加**します。
  - これにより、リンクバッジがあれば音声はその左に、なければ音声が右端に配置されます。
  - 音声バッジのデザインをリンクバッジ（h-8, rounded-full, 影付き）に合わせます。
- [MODIFY] コンテンツエリア内の `AudioPlayer` 表示を削除（またはオプション化）。
  - バッジ化により、中央の大きな再生ボタンは不要になるため削除します。

### 2. StudyCard.tsx (学習画面)
- [MODIFY] `extraHeaderLeft` で渡している音声ボタンを削除。`Flashcard.tsx` 側の内部ロジックで音声を自動検知してバッジを表示するように統一します。

### 3. デザイン仕様の統一
- **リンクバッジ (既存)**: `h-8`, `rounded-full`, `bg-primary-600` (またはテーマ色), アイコン `LinkIcon`.
- **音声バッジ (新規)**: `h-8`, `rounded-full`, `bg-primary-600` (または同等の色), アイコン `Volume2` または `Play`.

## 検証計画

### 動作確認
- **Flashcard**: プレビューモードおよび通常表示で、音声バッジが右下に表示されるか確認。
- **StudyCard**: 学習画面の右下に音声バッジとリンクバッジが正しく並んでいるか確認。
- **CardList**: カード一覧画面（エクスプローラー）でのアイコン配置も必要に応じて調整されているか確認。

### レイアウト確認
- 音声マークがリンクマークの「左側」にあること。
- 両者の高さ・形状が一致していること。
