# 統計画面のデザイン改善 実装計画

統計画面のデザインを、他の主要画面（ダッシュボードやフォルダ管理画面）と統一し、より洗練された印象に改善します。

## 改善のポイント
- **背景の統一**: 統計画面独自のグラデーションを廃止し、アプリ共通の背景色（`#F8FAFB`）に統一します。
- **ヘッダーの再設計**: フォルダ一覧などの画面と親和性の高い、カード型のヘッダーまたはアイコン装飾付きのヘッダーに変更します。
- **カードの一貫性**: グラフやパネルに使用するカードの角丸（`rounded-[32px]`相当）やシャドウ、余白を統一します。
- **アルゴリズムパネルの洗練**: 「計算ロジック」セクションを、よりモダンな情報パネルのUIにブラッシュアップします。
- **配色とフォントの統一**: 他の画面で使用されている `slate` カラーや `primary` カラーのトーン、フォントウェイトを揃えます。

## 変更内容

### 1. ページ全体のレイアウト ([Statistics.jsx](file:///c:/FlashcardMaster/src/Pages/Statistics.jsx))
- 背景色を `bg-[#F8FAFB]` に変更。
- メインコンテンツの最大幅とパディングを `max-w-[1400px] mx-auto p-4 md:p-14` で統一（維持しつつ微調整）。
- ヘッダー部分を `Folders.jsx` のようなカード形式、または `Dashboard.jsx` のようなアイコンボックス付きのスタイルに変更。

### 2. アルゴリズムパネル ([Statistics.jsx](file:///c:/FlashcardMaster/src/Pages/Statistics.jsx))
- `AlgorithmPanel` のデザインを、メインの統計カードと調和するように調整。
- ステータス（忘れた、OKなど）の表示を、よりコンパクトで清潔感のあるデザインに変更。

### 3. グラフコンポーネント ([StatsCharts.tsx](file:///c:/FlashcardMaster/src/Components/stats/StatsCharts.tsx))
- `StabilityDistributionChart` のカードスタイルを `rounded-[32px]` に統一。
- グラフ内部の配色やフォントサイズを微調整し、視認性を向上。

## 検証プラン

- [ ] 統計画面を開き、背景色が他の画面（フォルダ一覧など）と同じであることを確認。

- モバイル表示（画面幅を狭めた状態）でのレイアウト崩れがないか確認します。
