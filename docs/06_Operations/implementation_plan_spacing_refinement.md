# ブロック余白改善計画

ユーザーからのフィードバックに基づき、カードブロックの外枠と内部コンテンツ（コードブロック等）の間の余白（白いスペース）を削減します。

## 修正内容

### 1. `src/Components/card/blocks/BlockWrapper.tsx` の修正

ブロック外枠のパディングを大幅に削減し、コンテンツを枠一杯に広げます。

- **Padding**: `p-6 md:p-8` → `p-1.5` (約6px) に変更。
- これにより、コードブロックや画像アップロードエリアが枠のデザインと一体化し、二重枠のような隙間感を解消します。

### 2. `src/Components/card/blocks/TextBlock.tsx` の修正

`BlockWrapper` のパディング削除に伴い、テキスト入力エリアが枠に接近しすぎるため、テキストブロック側でパディングを確保します。

- **Padding**: `px-4 py-2` (または適切な値) を追加し、読みやすさを維持します。

## 検証計画

### 目視確認

- コードブロック等のヘッダーが外枠の角丸に沿って配置され、不自然な白い隙間がなくなっていること。
- テキストブロックの入力文字が枠に張り付かず、適切な余白が保たれていること。
- 全体的にブロックが「ぎゅっと」引き締まった印象になっていること。
