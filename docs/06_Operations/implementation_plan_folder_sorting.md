# フォルダ並び替え機能の実装計画

フォルダ詳細画面（FolderView）において、サブフォルダの並び順を改善し、ユーザーが並び替えを行えるようにします。

## 概要

- サブフォルダのデフォルトの並び順を「作成日時の新しい順」に変更します。
- サブフォルダの並び替え（作成日時昇順/降順、名前順など）を選択できるUIを追加します。

## 変更内容

### 1. [FolderView.jsx](file:///c:/FlashcardMaster/src/Pages/FolderView.jsx)

- **状態管理の追加**:
  - `folderSortMode` 状態を追加（初期値: `'created_desc'`）。
- **ソートロジックの実装**:
  - `childFolders` を `folderSortMode` に基づいてソートする `sortedChildFolders` を `useMemo` で作成します。
- **UIの追加**:
  - 「階層構造 (SUB FOLDERS)」セクションの横に、並び替えオプションを選択できるドロップダウンメニューを追加します。
  - カードの並び替えUIと同様のスタイルを採用し、一貫性を保ちます。

### 2. 並び替えオプション

以下のオプションを提供します：

- **作成日 (新しい順)**: `created_desc` (デフォルト)
- **作成日 (古い順)**: `created_asc`
- **名前 (昇順)**: `name_asc`
- **名前 (降順)**: `name_desc`

## 検証プラン

### 自動テスト
- `npm run build` を実行し、構文エラーがないことを確認します。

### 手動検証

1. **初期表示の確認**: フォルダ詳細画面を開いた際、サブフォルダが新しく作成された順に並んでいることを確認します。
2. **並び替えの変更**: ドロップダウンから別のソート順を選択し、フォルダの並びが正しく更新されることを確認します。
3. **新規作成時の挙動**: 新しいフォルダを作成した際、（作成日降順であれば）リストの先頭に表示されることを確認します。

## 注意事項
- 現時点では、サブフォルダのドラッグ＆ドロップによる手動並び替え（custom）は実装の複雑さを考慮し、まずは属性ベースのソートから提供します。ユーザーの要望が「作成順」および「並び替えができること」であったため、まずはこれらを優先します。
