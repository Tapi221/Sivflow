# フォルダ一覧のボタン表示制御 (ホバー時のみ表示)

## 概要
フォルダ行の更なるシンプリシティを追求するため、「＋ (カード作成)」と「︙ (メニュー)」ボタンをデフォルトで非表示にし、フォルダ行にマウスホバー（またはタップ）した時のみ表示されるように調整します。

## 変更内容

### Components/folder

#### [MODIFY] [FolderTree.tsx](file:///c:/FlashcardMaster/src/Components/folder/FolderTree.tsx)

- **アクションボタンの表示制御**:
  - 右側のボタンのコンテナ (`div`) に対して、`opacity-0 group-hover:opacity-100 transition-opacity duration-200` を適用します。
  - すでに親要素には `group` クラスが付与されているため、これを利用します。
  - モバイル対応（タップ時）についても、多くのモバイルブラウザではタップが hover 状態として扱われるため、基本的には同じクラスで対応可能です。もし不十分な場合は `active:opacity-100` 等の追加を検討します。

- **プレイスホルダーの維持**:
  - `opacity-0` を使用することで、ボタンが「存在はしているが透明」な状態にし、ホバー時にガタつき（レイアウトシフト）が発生しないようにします。

## 検証計画
- マウスホバー時にのみボタンが現れることを確認。
- ボタンがない状態でも、タイトルやメタデータが正しく配置されているか確認。
- モバイルの実機相当のモードで、フォルダ行をタップした際にボタンが表示されるか確認。
