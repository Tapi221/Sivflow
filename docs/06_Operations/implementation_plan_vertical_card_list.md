# カード詳細リストモードの実装計画

## 概要
「CARD」タブにおいて、カードの情報を網羅的に表示する縦並びのリストモードを追加します。
現在のグリッド表示（要約表示）に加え、質問・解答の全文を確認できる詳細リスト表示を切り替えられるようにします。

## 変更内容

### 1. `src/Pages/FolderView.jsx`
-   **State追加**: `viewMode` (`'grid' | 'list'`) を追加（初期値 `'grid'`）。
-   **UI追加**: ソートボタンの横（またはツールバー内）に表示切り替えボタン（アイコン: `LayoutGrid` / `List`）を追加。
-   **Props変更**: `CardList` コンポーネントに `viewMode` を渡す。

### 2. `src/Components/card/CardList.tsx`
-   **Props追加**: `viewMode` を受け取る。
-   **レイアウト変更**:
    -   `viewMode === 'list'` の場合、コンテナのクラスを `grid-cols-1` に固定し、最大幅を制限（読みやすさのため `max-w-3xl mx-auto` 等）。
-   **表示コンポーネントの拡張**:
    -   `CardItem` (または新規 `VerticalCardItem`) で、`viewMode === 'list'` の場合のレンダリングロジックを追加。
    -   **リストモード時の表示項目**:
        -   ヘッダー: 既存のステータスアイコン等は維持。
        -   **コンテンツ**:
            -   **Question**: 全文表示 (`line-clamp` なし)。
            -   **Answer**: 全文表示 (`line-clamp` なし)。
            -   **Blocks**: テキスト以外のブロック（画像、コード、数式など）が含まれる場合、その種別と数をアイコンで表示（例: 📷 画像 x 2）。
        -   レイアウト: 縦方向に余裕を持った配置。

## 検証計画
1.  **表示切り替え**:
    -   フォルダ詳細画面で表示切り替えボタンをクリックし、グリッド表示とリスト表示が切り替わることを確認。
2.  **リスト表示の内容**:
    -   長いテキストを持つカードが省略されずに表示されること。
    -   質問と解答の両方が表示されること。
    -   画像や音声を含むカードで、インジケータが表示されること。
3.  **動作確認**:
    -   リスト表示モードでも編集・削除・ドラッグ＆ドロップ（並び替え）が正常に機能すること。
