# カラム表示におけるタグ絞り込み機能の追加

カラム表示（Finderスタイル）において、タグでフォルダを絞り込む機能を追加します。
フォルダ自体にはタグがありませんが、カードにはタグがあるため、「選択したタグを持つカードを含むフォルダ（およびその親フォルダ）」を表示するロジックで実装します。

## ユーザーレビューが必要な事項
- タグ絞り込み時の挙動：
    - 選択したタグを持つカードが1つでも含まれるフォルダ、およびその祖先フォルダのみが表示されます。
    - タグを複数選択した場合は、いずれかのタグ（OR条件）を持つカードを対象とします。

## 提案される変更点

### フォルダ管理コンポーネント

#### [MODIFY] [ColumnNavigator.tsx](file:///c:/FlashcardMaster/src/Components/folder/ColumnNavigator.tsx)
- `selectedTags` 状態を追加。
- カラム表示の上部にタグ選択UIを追加。
- 選択されたタグに基づいて、表示すべきフォルダIDのセット（`visibleFolderIds`）を計算するロジックを追加。
- `FolderColumn` に `visibleFolderIds` を渡し、フィルタリングを指示。

#### [MODIFY] [FolderColumn.tsx](file:///c:/FlashcardMaster/src/Components/folder/FolderColumn.tsx)
- `visibleFolderIds` プロップスを受け取る。
- `childFolders` のフィルタリング時に、`visibleFolderIds` に含まれているかどうかをチェックする条件を追加。

## 検証計画

### 手動確認
1. フォルダ画面で「カラム」表示に切り替える。
2. 上部に追加されたタグ一覧から特定のカテゴリ（タグ）を選択する。
3. そのタグを持つカードが含まれるフォルダの階層パスだけが表示されることを確認。
4. タグを解除して、全フォルダが再表示されることを確認。
5. 複数のタグを選択し、OR条件で正しく絞り込まれることを確認。
