# FolderMemo.tsx のアクセシビリティ修正計画

## 概要
IDEで報告されたアクセシビリティエラー（ボタンのテキスト欠落、画像の代替テキスト欠落、フォームラベル欠落）を修正します。また、インラインスタイルに関する警告については、動的なCSS変数の設定であるため、現状維持としつつ必要ならコメントを追加します。

## 修正内容

### src/Components/folder/FolderMemo.tsx

1.  **画像の削除ボタン (Line 260付近)**
    - `button` 要素に `title="画像を削除"` と `aria-label="画像を削除"` を追加します。

2.  **ファイルアップロード入力 (Line 283付近)**
    - `input type="file"` 要素に `title="画像をアップロード"` と `aria-label="画像をアップロード"` を追加します。

3.  **メモリストの画像表示 (Line 473付近)**
    - `img` 要素に `alt="メモ画像"` (または `alt="Memo upload"` 等) を追加します。

## 検証計画

### 手動検証
1.  **修正箇所の確認**
    - フォルダメモを開き、画像のアップロード、削除ボタン、保存済み画像の表示が正常に行えるか確認（機能が壊れていないか）。
    - ブラウザの検証ツールで、該当要素に `title`, `aria-label`, `alt` 属性が付与されているか確認。
