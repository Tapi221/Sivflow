じ# フォルダ画面のGlassmorphism化 実装計画

## 目的
フォルダ一覧画面（`Folders.jsx`, `FolderView.jsx`を含む）のデザインを、指定された「Glassmorphism（ガラス）」スタイルに変更する。
既存の機能・ロジックは一切変更せず、CSS/Tailwindのみで見た目を刷新する。

## ユーザーレビューが必要な事項
- **デザイン変更の範囲**: 基本的に「フォルダ」に関連する画面（`/folders`, `/FolderView`）のみが対象だが、サイドバーはグローバルレイアウトに含まれるため、フォルダ画面表示時のみスタイルを上書きする実装となること。

## 変更内容

### 1. グローバルスタイル (`src/index.css`)
Glassmorphism用のユーティリティクラス（`@layer utilities`）を追加する。
これにより、他のコンポーネントでも「ガラス」スタイルを再利用可能にする。

- **背景**: `.bg-glass-navy` (Navy base + Gradients)
- **ノイズ**: `.bg-noise` (SVG Data URI pattern)
- **ガラスパネル**:
    - `.glass-panel-sidebar`: サイドバー用（薄めのガラス）
    - `.glass-panel-content`: メインコンテンツ用（中程度のガラス、blur強め）
    - `.glass-row`: フォルダリスト行用（薄いガラス、Hover効果あり）
- **テキスト**:
    - `.text-glass-high`: 白 (0.95)
    - `.text-glass-med`: 白 (0.75)
    - `.text-glass-low`: 白 (0.50)
- **ボタン**:
    - `.btn-glass`: ガラス風ボタン

### 2. レイアウト (`src/Layout.tsx`)
現在のページが `Folders` または `FolderView` の場合のみ、サイドバー (`aside`) のスタイルを条件付きで変更する。

- **変更前**: `bg-white border-r border-slate-50`
- **変更後**: 条件付きで `glass-panel-sidebar border-r-0` 等を適用。
- **テキスト色**: サイドバー内のリンクテキストやアイコンの色も、背景に合わせて「白ベース」に変更する（条件付きクラス）。

### 3. フォルダ一覧ページ (`src/Pages/Folders.jsx`)
- ルート要素 (`div`) に `bg-glass-navy bg-noise` を適用。
- ヘッダー (`Card`) の背景を削除し、`glass-panel-content` に置き換え。
- テキスト色を `.text-glass-*` 系に変更。
- パンくずリストやボタンのデザインをガラス風に調整。

### 4. フォルダ詳細ページ (`src/Pages/FolderView.jsx`)
- `Folders.jsx` と同様に、ルート背景とヘッダー、ツールバーをガラス化する。
- 既存の白い背景 (`bg-slate-50` 等) を削除。
- `FolderTree` に渡される props (e.g. `isGlassMode`) を通じて、子要素のスタイル変更を伝える（もしくは CSS Context で伝播させる）。

### 5. フォルダ行コンポーネント (`src/Components/folder/FolderTree.tsx`)
- `FolderItem` の `div` スタイルを修正。
- **変更前**: `bg-white border-slate-100 ...`
- **変更後**: 親からガラスモードが指定されている場合、`glass-row border-transparent text-glass-high ...` を適用。
- アイコン（Folder icon, Chevron, etc.）の色を白ベースに調整。

## 検証計画

### 自動テスト
- 今回はスタイル変更のみであり、既存のユニットテスト（ロジック確認用）はそのまま通過することを期待する。
- 新規の自動テストは作成しない（Visual Regression Test環境がないため）。

### 手動検証手順
1. アプリを起動し、ログインする。
2. 左サイドバーから「フォルダ」をクリックする。
3. **フォルダ一覧画面**のデザインを確認する。
    - [ ] 背景が「暗いネイビー」＋「グラデーションブロブ」＋「ノイズ」になっているか。
    - [ ] 左サイドバーが「ガラス（半透明）」になっているか。
    - [ ] 上部の操作バーが「ガラス」になっているか。
    - [ ] フォルダリストの各行が「薄いガラスカード」になっているか。
    - [ ] テキスト（フォルダ名、カード数など）が白文字で読みやすいか。
4. **ホバー動作**を確認する。
    - [ ] フォルダ行にマウスを乗せた時、少し明るくなる/影が濃くなるか。
5. **フォルダ詳細画面**を確認する。
    - [ ] フォルダをクリックして詳細画面 (`/FolderView`) に遷移する。
    - [ ] 同様に背景・ヘッダーがガラス化されているか確認する。
6. **他画面への影響なし**を確認する。
    - [ ] 「学習 (Dashboard)」や「統計 (Statistics)」をクリック。
    - [ ] サイドバーや背景が元の「白ベース」に戻ることを確認する。
