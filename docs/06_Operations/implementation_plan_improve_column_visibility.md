# カラム表示の視認性改善計画

カラム表示（Finderスタイル）において、「色が薄くて見づらい」という問題を解決するため、テキストのコントラスト向上、選択状態の強調、および境界線の明確化を行います。

## 変更内容

### フォルダカラムコンポーネント

#### [MODIFY] [FolderColumn.tsx](file:///c:/FlashcardMaster/src/Components/folder/FolderColumn.tsx)
- **カラム境界線の強化**:
    - `border-r-slate-100` を `border-r-slate-200` に変更し、カラムの区切りを明確にします。
- **ヘッダーの強調**:
    - 背景色 `bg-slate-50/50` を `bg-slate-100/50` に、テキスト `text-slate-400` を `text-slate-500` に変更し、各階層の見出しを読みやすくします。
- **フォルダ・カード名のコントラスト向上**:
    - 未選択時のテキスト `text-slate-600` を `text-slate-700` （または `slate-800`）に変更します。
    - カード数などの補足テキスト `text-slate-400` を `text-slate-500` に変更します。
- **カード枚数表示の色の統一**:
    - カラム表示のカード枚数の色を、ツリー表示と同様の `text-primary-600` (または `primary-700/70`) に変更し、スタイルを統一します。
- **選択状態の視認性向上**:
    - 選択時の背景色とテキスト色が確実に高いコントラストを保持するように調整します。現在 `primary-500` ですが、テーマによって薄い場合は調整を検討します。
- **アイコンとインジケーター**:
    - 右側の `ChevronRight` インジケーターの色を `text-slate-300` から `text-slate-400` へ濃くします。
    - 未選択時のアイコン背景 `bg-slate-100` を少し濃くするか、アイコン自体の色を強調します。

### カラムナビゲーター

#### [MODIFY] [ColumnNavigator.tsx](file:///c:/FlashcardMaster/src/Components/folder/ColumnNavigator.tsx)
- **全体の枠線の強化**:
    - 外枠の `border-slate-200/60` を `border-slate-300/50` など、よりはっきりとした色に変更します。

## 検証計画

### 手動確認
1. フォルダ管理画面でカラム表示に切り替える。
2. 各カラムの境界線がはっきりと見えることを確認する。
3. フォルダ名やカード名が背景と区別しやすくなっていることを確認する。
4. フォルダを選択した際、選択されたアイテムが周囲から際立って見えることを確認する。
5. ダークモード（もしあれば）でも視認性が損なわれていないことを確認する。
