# カードエディタヘッダーレイアウト変更とドラッグ制限の実装完了報告

## 変更内容の概要

### 1. カードエディタヘッダーのレイアウト変更 (`CardEditor.tsx`)
*   **配置変更**: 
    *   「INDEX Q◯」ロゴのすぐ右側に、ブックマークアイコンと不確実フラグ（はてなマーク）アイコンを縦並びで配置しました。
    *   これらのアイコンを右側のツールバーから移動・削除しました。
*   **目的**: 重要なステータス切り替えにアクセスしやすくし、視線移動を減らすため。

### 2. OneQAモードのドラッグ移動制限 (`OneQAMode.jsx`)
*   **方向制限**: 
    *   カードの並び替え（ドラッグ＆ドロップ）時の移動方向を**上下（Y軸）のみ**に制限しました。
*   **目的**: ドラッグ中にカードが横方向にずれるのを防ぎ、リスト操作の快適性を向上させるため。

## 検証方法

### 1. ヘッダーレイアウトの確認
1.  任意のカードを開き、編集画面に入ります。
2.  左上の「INDEX Q番号」の右横を確認してください。
3.  **上**: ブックマークアイコン
    **下**: はてなマーク
    が縦に並んでいることを確認してください。
4.  アイコンをクリックして、正しくステータスがオン/オフ（色が変わる）することを確認してください。
5.  PC表示時、右側の「下書き」スイッチ付近に重複してアイコンが表示されていないことを確認してください。

### 2. ドラッグ移動の確認 (OneQAモード)
1.  「一問一答モード」を開始し、カードを2枚以上作成してください。
2.  カード右上のグリップアイコンを掴んでドラッグしてください。
3.  マウスを左右に大きく振っても、カードが**垂直線上**から外れずに上下にのみ移動することを確認してください。
4.  ドロップした位置で正しく並び替えが反映されることを確認してください。
