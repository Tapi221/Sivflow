# リンクブロックの余白削減計画

リンクブロック内の白い余白が多すぎるため、パディングやギャップを調整してコンパクトにします。

## 変更内容

### 1. [ReferenceBlock.tsx](file:///c:/FlashcardMaster/src/Components/card/blocks/ReferenceBlock.tsx)
- **全体パディングの削減**: 
    - `BlockWrapper` 内の `div` (line 57) の `p-4` を `p-2` か `p-3` に変更します。
- **カード内パディングの削減**:
    - 各リンクアイテムのコンテナ (line 59) の `p-4` を `p-3`（または `p-3 pb-4` など）に変更します。
- **フィールド間ギャップの調整**:
    - URLと表示名の間の `gap-3` (line 68) を必要に応じて `gap-2` に微調整します。
- **ラベルの配置**:
    - ラベルの `space-y-1` (line 69, 78) が縦方向の余白を生んでいるため、これを `space-y-0.5` にするか、ラベル自体のマージンを減らします。

## 検証計画

### 手動検証
1. 編集画面でリンクブロックを表示する。
2. 他のブロック（テキスト、画像など）と比較して、余白が適切に締まっているか確認する。
3. モバイル表示（レスポンシブ）でも入力欄が狭くなりすぎていないか確認する。
