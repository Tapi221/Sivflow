# 同期ステータスバーのガラスデザイン化

## 目的
同期ステータスや最終同期時間を表示するバーを、現在の不透明なデザインからガラス仕様（グラスモルフィズム）に変更し、UIの透明感とプレミアム感を向上させます。

## 提案される変更点

### 同期コンポーネント

#### [MODIFY] [SyncStatusIndicator.tsx](file:///c:/FlashcardMaster/src/Components/sync/SyncStatusIndicator.tsx)
- **コンテナのスタイル変更**:
    - 全体を包む `div` (またはその内部のラッパー) に以下のスタイルを適用します。
        - `bg-white/40` (または `bg-white/60`) : 半透明の白背景
        - `backdrop-blur-xl` : 背景のぼかし効果
        - `border border-white/40` : ガラスの厚みを感じさせる繊細な境界線
        - `shadow-[0_8px_32px_0_rgba(31,38,135,0.07)]` : 柔らかい浮遊感のある影
        - `rounded-full` : 丸みを帯びた形状（画像に合わせて）
- **内部要素の調整**:
    - ボタンやテキストの余白・パディングを画像のレイアウトに合わせて微調整。
    - アイコンやテキストの色が、半透明背景の上でも視認性が高くなるように調整。

## 検証計画

### 手動確認
1. アプリケーションの下部（またはヘッダー）にある同期バーを確認する。
2. 背景が透過してぼけており、境界線が白く光っていることを確認。
3. 状態の変化（同期中、成功、エラー）に応じて適切なアイコンとテキストが表示されることを確認。
4. 手動同期ボタンやメニューが引き続き正常に動作することを確認。
