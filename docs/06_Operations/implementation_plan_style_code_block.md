# コードブロック表示のスタイル改善

ユーザーの要望に基づき、コードブロック（`CodeRenderer`）の言語表示部分の縦幅を縮小し、インデントを視覚的に分かりやすくするための変更を行います。

## 変更内容

### 1. 言語表示ヘッダーの縦幅縮小
- **対象ファイル**: `src/Components/card/CodeRenderer.tsx`
- **変更点**:
  - ツールバーのパディングを `py-1.5` から `py-0.5` に変更し、高さを圧縮します。
  - 必要に応じて高さを明示的に `h-6` 程度に設定し、`CodeBlockEditor` との視覚的な統一感を図ります。

### 2. インデントの可視化
- **対象ファイル**: `src/Components/card/CodeRenderer.tsx` (または `src/index.css`)
- **変更点**:
  - コード表示領域（`pre`タグ）に、インデントガイドとなる縦線を表示するスタイルを追加します。
  - CSSの `background-image` と `background-size` を使用し、`2ch`（2文字分）ごとのグリッド線を描画します。
  - `background-attachment: local` を設定し、スクロールに追従するようにします。

#### 具体的なCSS実装イメージ
```css
background-image: linear-gradient(to right, transparent 0%, transparent calc(100% - 1px), #e2e8f0 100%);
background-size: 2ch 100%; /* 2文字幅ごとに線を引く */
background-attachment: local;
```
※ 色は `slate-200` (`#e2e8f0`) などを調整して、目立ちすぎないようにします。

## 検証計画

### 自動テスト
- 現在、UIの見た目に関する自動テストは存在しないため、新たな自動テストは作成しません。

### 手動検証
1. **設定画面** または **カード編集画面**（プレビュー）にて、コードブロックを含むカードを表示します。
2. **言語ヘッダーの確認**:
   - ヘッダー部分の高さが以前より狭くなっていることを目視で確認します。
3. **インデントガイドの確認**:
   - インデント（スペース）が含まれるコードを表示し、薄い縦線が2文字間隔（または設定した間隔）で表示されていることを確認します。
   - 長い行がある場合、横スクロールしてもガイドが追従することを確認します。
4. **ダークモード確認**（もしあれば）:
   - ガイドの色が適切に見えるか確認します（今回はライトモードメインで調整）。

## 実行手順
1. `src/Components/card/CodeRenderer.tsx` を修正。
2. 必要であれば `src/index.css` にユーティリティクラスを追加（今回はインラインスタイルまたはTailwindクラスで完結予定）。
