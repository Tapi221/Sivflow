# 実装計画：階層別フォルダ作成ボタンの追加

フォルダ管理画面において、各階層（ルートおよび展開された子階層）の最下部に、新規フォルダ作成ボタンを配置します。

## 要件

- フォルダアイテムと同じサイズ・レイアウトで表示
- グレーのシルエット風デザイン（破線ボーダー、薄い背景）
- クリックすると、その階層に新しいフォルダーを作成
- ルート階層と各子階層の両方に配置

## 変更内容

### [FolderTree](file:///c:/FlashcardMaster/src/Components/folder/FolderTree.tsx)

1. **ルート階層の作成ボタン**
   - `rootFolders` の後に、新規フォルダ作成ボタンを追加
   - `onCreateFolder(null)` を呼び出してルートフォルダーを作成

2. **子階層の作成ボタン**
   - 各 `FolderItem` の子フォルダーリストの後に、作成ボタンを追加
   - `onCreateSubfolder(folder.id)` を呼び出して子フォルダーを作成

3. **デザイン**
   - 既存のフォルダーアイテムと同じクラス構造を使用
   - `bg-slate-50` で薄いグレー背景
   - `border-dashed border-slate-300` で破線ボーダー
   - アイコンは `FolderPlusIcon` を使用
   - テキストは「新規フォルダ」
   - ホバー時に `hover:bg-slate-100 hover:border-slate-400` で視覚的フィードバック
