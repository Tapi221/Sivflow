# 一問一答モード初期化およびフォーカス最適化 実装レポート

## 概要
一問一答モードにおいて、ユーザーがカード作成をスムーズに開始できるよう、初期状態で2枚のカードを表示し、タイトル設定に応じて適切な位置にフォーカスを当てる改善を行いました。

## 変更内容

### 1. 初期カード数の変更
- `src/Pages/OneQAMode.jsx` において、`editors` の初期状態を1枚から2枚に変更しました。
- 1枚目のカードにのみ `autoFocus: true` を設定し、2枚目は表示のみ行うようにしました。

### 2. フォーカス制御の伝播
- **CardEditor.tsx**: `hideTitle` プロップが `true` の場合、タイトル入力欄へのフォーカスをスキップし、問題セクションの `BlockEditor` に `autoFocus` を渡すようにしました。
- **BlockEditor.tsx**: `autoFocus` プロップを受け取り、最初のブロック（`index === 0`）が `TextBlock` の場合に `autoFocus` プロップを渡すようにしました。
- **TextBlock.tsx**: 受け取った `autoFocus` プロップを `AutoResizeTextarea` の `autoFocus` 属性に適用しました。

## 確認事項
- [x] 一問一答モード遷移時に初期状態で2枚のカードが表示される。
- [x] タイトルあり時：1枚目のタイトルにフォーカスが当たる。
- [x] タイトルなし時：1枚目の問題文（テキストブロック）にフォーカスが当たる。
- [x] 保存後の「続けて作成」時も、新しく追加されたカードに正しくフォーカスが当たる（既存機能の維持）。
