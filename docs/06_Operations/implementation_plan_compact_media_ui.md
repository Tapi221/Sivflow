# 実装計画：画像・音声ブロックのコンパクト化

ユーザーからの要望に基づき、カードエディタ内の画像ブロックおよび音声ブロックの縦幅を縮小し、より多くのコンテンツを一度に表示できるように改善します。

## 変更内容

### [UI/UX] Components/card/[MediaUploader.tsx](file:///c:/FlashcardMaster/src/Components/card/MediaUploader.tsx)

空の状態のアップロードエリア、およびコンテンツがある状態のUIをよりコンパクトに調整します。

#### 未アップロード（空）の状態
- 外側のパディングを縮小: `p-8` → `p-5`
- アップロードアイコンを縮小: `w-8 h-8` → `w-6 h-6`
- アイコン下のマージンを縮小: `mb-3` → `mb-1.5`
- テキストの改行を削除し、1〜2行に収まるように調整
- 全体の角丸は維持しつつ、視覚的な圧迫感を軽減

#### アップロード済みの状態
- 画像プレビュータイルの高さを縮小: `h-32` → `h-24`
- 「追加」ボタンタイルの高さを縮小: `h-32` → `h-24`
- 音声追加ボタンのパディングを縮小: `py-4` → `py-2.5`

## 確認事項
- [ ] 画像アップロードが正常に動作し、コンパクトなプレビューが表示されること
- [ ] 音声アップロードが正常に動作し、コンパクトな追加ボタンが表示されること
- [ ] ドラッグ＆ドロップ領域が適切に反応すること
- [ ] モバイル表示でレイアウトが崩れないこと
