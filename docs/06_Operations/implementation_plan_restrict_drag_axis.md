# OneQAモードのドラッグ移動方向制限

## 概要
OneQAモード（一問一答モード）におけるカードの並び替え操作において、ドラッグ移動を上下方向（Y軸）のみに制限します。これにより、横方向への意図しないズレを防ぎ、リストの操作性を向上させます。

## 変更内容

### `src/Pages/OneQAMode.jsx`

*   `<Draggable>` コンポーネントに `lockAxis="y"` プロパティを追加します。
    *   これにより、ユーザーがカードをドラッグした際、マウスカーソルが左右に動いてもカードは垂直軸上に固定されて移動します。

```jsx
<Draggable 
  key={editor.id} 
  draggableId={editor.id} 
  index={index}
  lockAxis="y" // 追加
>
```

## 検証方法

### 手動検証
1.  **OneQAモードの起動**:
    *   任意のフォルダから「一問一答モード」を開始する、またはURLパラメータでOneQAモードを開く。
    *   カードを複数枚追加する（「続けて作成」または自動追加）。
2.  **ドラッグ操作**:
    *   カード右上のグリップアイコン（またはカード全体）を掴んでドラッグする。
    *   マウスを左右に大きく振ってみる。
3.  **確認事項**:
    *   カードが左右には移動せず、上下方向にのみ追従することを確認する。
    *   並び替えが正常に機能することを確認する。
