# サブフォルダ表示の折りたたみ機能 完了報告

フォルダ詳細画面において、サブフォルダ（階層構造）の表示を初期状態で1行に制限し、展開/折りたたみができる機能を実装しました。

## 実施内容

### 1. 状態管理の追加
- `isFoldersExpanded` 状態を導入し、デフォルトを `false`（折りたたまれた状態）に設定しました。

### 2. UIの更新 (`FolderView.jsx`)
- 「階層構造 (SUB FOLDERS)」セクションの右側に「すべて表示 / 折りたたむ」ボタンを追加しました。
- ボタンには現在のフォルダ総数を表示し、アイコン（ChevronDown/Up）で状態を示します。

### 3. レスポンシブな制限ロジック
- CSS の `max-height` と `overflow-hidden` を組み合わせることで、JavaScript で表示数を計算することなく、画面幅に応じた「1行」の表示を実現しました。
- `transition-all` により、展開/折りたたみ時にスムーズなアニメーションが適用されます。

## 動作イメージ
- **折りたたみ時**: 画面幅に合わせて 1行（PCなら4つ、モバイルなら1つ）のみが表示されます。
- **展開時**: 全てのサブフォルダが表示されます。
