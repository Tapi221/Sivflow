# カード編集画面 レイアウト調整・余白削減計画

## 目標
カード編集画面のヘッダー部分をよりコンパクトかつ機能的にするため、タイトル入力とタグ入力を横並びに配置し、不要な余白を削減する。

## 変更内容

### `src/Components/card/CardEditor.tsx`

1.  **タイトルとタグの横並び化**:
    - タイトルとタグを包むコンテナを `flex flex-col md:flex-row` に変更し、横並びに対応させる。
    - タイトル側の幅を広め (`flex-[2]`)、タグ側をやや狭め (`flex-[1]`) に配分する、あるいは柔軟に調整する。

2.  **余白の削減**:
    - 各要素間の `gap` を縮小する（例: `gap-5` -> `gap-3`）。
    - ラベル部分の余白（`mb-1` など）をよりタイトにする。
    - セクション全体の `px-4` や `py-x` を見直し、よりスリムな印象にする。

3.  **ラベルのコンパクト化**:
    - ラベルテキストのサイズやトラッキングを微調整し、占有面積を減らす。

### 修正イメージ（クラス名等）
- タイトル・タグ用親コンテナ: `flex flex-col md:flex-row gap-4 w-full`
- タイトルセクション: `flex-1 md:flex-[2]`
- タグセクション: `flex-1 md:flex-[1]`

## 検証計画

### 手動検証
1.  **デスクトップ**:
    - タイトルとタグが1行に収まっていることを確認。
    - 余白が詰まったことで、メインの入力エリア（問題・解答）がより上部に表示されることを確認。
2.  **モバイル**:
    - 画面幅が狭い場合は、従来通り縦にスタックされ、崩れがないことを確認。
3.  **入力操作**:
    - タイトル入力、タグ追加の操作性が低下していないことを確認。
