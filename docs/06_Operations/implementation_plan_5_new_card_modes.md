# 実装計画：カード表示バリエーションの追加（5案）

既存の表示モード（グリッド、リスト、コンパクト、ギャラリー、タイムライン）に加え、新たに5つの表示スタイルを実装します。これにより、用途や好みに合わせた最適な学習環境を提供します。

## 追加する5つのモード

1. **Table (テーブル)**
   - スプレッドシート形式。タイトル、定着度、タグ、更新日などを一列で表示。
   - 大量のカードを効率的に管理・比較するのに最適。

2. **Hero (ヒーロー)**
   - 1枚を大きく、詳細まで表示。画像や解説テキストを強調。
   - じっくりと内容を確認したい場合に最適。

3. **Magazine (マガジン)**
   - タイポグラフィを重視し、ニュースサイトのようなレイアウト。
   - 視覚的な訴求力が高く、読みやすさを重視。

4. **Sticky (付箋)**
   - カラフルな付箋を並べたようなデザイン。少し傾けたり、ピン留めアイコンを表示。
   - 直感的で親しみやすい外観。

5. **Bullet (バレット)**
   - 極限まで要素を削ぎ落とした、アイコンとタイトルのみの1行表示。
   - 最も高密度な情報表示モード。
