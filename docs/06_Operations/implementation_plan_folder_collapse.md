# サブフォルダ表示の折りたたみ機能実装計画

フォルダ詳細画面において、サブフォルダ（階層構造）の表示を初期状態で1行に制限し、ユーザーが操作することで全件表示できるように変更します。

## 修正内容

### 1. 状態管理の追加
`FolderView.jsx` に `isFoldersExpanded` 状態を追加します。初期値は `false` とします。

### 2. 表示するフォルダ数の制限
`sortedChildFolders` のうち、画面のグリッド幅（1行分）に合わせて表示件数を動的に制限します。
- `isFoldersExpanded` が `true` の場合は全件表示。
- `false` の場合は、1行に収まる数（最大 3件 + 新規フォルダボタン = 合計4件）を表示します。
  - レスポンシブ対応を考慮し、画面幅に応じた制限値を設定します。

### 3. モアボタン（展開/折りたたみ）の追加
サブフォルダが1行に収まらない場合に、表示を切り替えるためのボタンを追加します。
- ボタンは「階層構造 (SUB FOLDERS)」見出しの右側、またはグリッドの最後に追加します。
- アイコン（ChevronDown / ChevronUp）を使用して視覚的に分かりやすくします。

## 検証計画

### 手動確認
- サブフォルダが多数あるフォルダを開き、初期状態で1行（4つ分）のみ表示されていることを確認する。
- 展開ボタンをクリックすると全件表示されることを確認する。
- 再度クリックすると折りたたまれることを確認する。
- サブフォルダが少ない（1行に収まる）場合に、展開ボタンが表示されないことを確認する。
- モバイル画面で1行（ボタン1つ分）に制限されているか確認する。
