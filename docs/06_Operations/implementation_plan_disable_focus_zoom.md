# テキスト入力時のフォントサイズ自動変更の無効化

## 目的
テキストブロックなどの入力時（フォーカス時）に、フォントサイズが勝手に変更（拡大/縮小）される挙動を廃止します。
ユーザーからの「入力時のみ文字サイズが変わるのをやめてほしい」という要望に対応します。

## 原因特定
`src/index.css` に以下の記述があり、フォーカスするだけでフォントサイズが `16px` に強制されています。
これにより、`16px` 以外のサイズ（`text-sm` や `text-2xl` など）が設定されている要素が、入力中のみ `16px` に変化してしまっています。

```css
/* src/index.css L63-66 */
input:focus, textarea:focus {
  font-size: 16px;
  scroll-margin-top: 100px;
}
```

## 変更内容

### 1. `src/index.css` の修正
- `input:focus, textarea:focus` ルールセットから `font-size: 16px;` を削除します。
- `scroll-margin-top` はスクロール位置の調整のために有用な可能性があるため維持します（もし要望があれば削除しますが、今回はサイズ変更が主眼ですので維持を基本とします）。

```diff
  /* 入力フォーカス時のズーム防止とジャンプ防止 */
  input:focus, textarea:focus {
-   font-size: 16px;
    scroll-margin-top: 100px;
  }
```

## 検証手順
1. アプリケーションをビルド・起動する。
2. テキストブロック（Serifフォントの大きなテキストエリア）をクリックしてフォーカスする。
3. フォーカス時に文字サイズが変わらないことを確認する。
4. メモブロック（通常小さいテキスト）やタイトル入力欄でも、フォーカス時にサイズが変わらないことを確認する。
