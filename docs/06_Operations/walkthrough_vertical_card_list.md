# カードリスト 縦並び詳細モード実装完了

## 概要
カードリスト（CARDタブ）に「リスト表示モード」を追加しました。
従来のグリッド表示と切り替えることで、カードの質問・解答の重要部分を省略せずに確認できるようになりました。

## 実装内容
1.  **表示切り替え機能**:
    -   `FolderView` ヘッダー（ソートボタン付近）に表示切り替えトグルボタンを追加。
    -   「グリッド表示（要約）」と「リスト表示（詳細）」を選択可能。

2.  **リスト表示レイアウト**:
    -   カードを縦一列に配置し、幅を最適化（最大幅 `max-w-3xl`）。
    -   テキストの省略表示（`line-clamp`）を解除し、長文も全文表示。
    -   「ANSWER」セクションを明示的に表示し、質問と解答を区別しやすくレイアウト調整。

## 検証結果
-   [x] トグルボタンによる表示切り替えがスムーズに動作することを確認。
-   [x] リスト表示時に質問・解答が全文表示されることを確認。
-   [x] グリッド表示に戻した際に、従来通りのレイアウトが維持されていることを確認。
