# 実装計画：フォルダーツリー最下層の余白削除

フォルダー管理画面において、子フォルダーを持たない（最下層の）フォルダーに表示されているトグルアイコン用の空白スペースを削除し、見た目をスッキリさせます。

## 課題
- 子フォルダーがない場合でも、`<span className="w-2.5" />` によってトグルアイコン分のスペースが確保されており、不自然な空白が生じている。
- ユーザーはトグルアイコンがない場合にこの空白を詰めることを希望している。

## 変更内容

### [FolderTree](file:///c:/FlashcardMaster/src/Components/folder/FolderTree.tsx)
- `FolderItem` コンポーネント内のトグルボタン部分を修正。
- `hasChildren` が `false` の場合、プレースホルダーの `span` を表示しないように変更。
- または、ボタン自体の表示条件を見直し、アイコンがない場合にスペースを占有しないように調整する。

## 検証計画
- [ ] 子フォルダーを持つフォルダーには、引き続きトグルアイコン（Chevron）が表示されることを確認。
- [ ] 子フォルダーを持たないフォルダーでは、トグルアイコン用の空白がなくなり、ドラッグハンドルとフォルダーアイコンが近づいていることを確認。
- [ ] ネストされた階層でも、一貫した見た目になっていることを確認。
