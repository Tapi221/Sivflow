# カードエディタレイアウト変更・ブロックツール縮小計画

## 概要
ユーザーの要望に基づき、カード編集画面のヘッダーレイアウトを再構築し、ブロックエディタの操作ツール（右側の複製・削除ボタン等）の見た目をヘッダーアイコンに合わせて縮小・最適化します。

## 変更内容

### 1. `src/Components/card/CardEditor.tsx`
*   **ヘッダー要素の並び順変更**:
    *   **現在**: `[Index/Q番号 + アイコン] [耐性スコア]`
    *   **変更後**: `[Index/Q番号] -> [耐性スコア] -> [アイコン(縦)]` の順に配置します。
    *   これらを一つの `flex` コンテナ内に並べ、デスクトップ表示 (`md`) では横一列、モバイルでは適切な折り返しまたは配置になるよう調整します。
*   **サイズ/配置調整**:
    *   「耐性スコア」の高さ・表示位置を、左隣の「Index Q番号」の表示サイズに合わせるよう調整します（`items-end` やフォントサイズ調整など）。
    *   「アイコン（ブックマーク/はてな）」も同様に、高さを揃えてその右に配置します。
    *   タイトル入力欄がその右側に続くレイアウトにします。

### 2. `src/Components/card/blocks/BlockWrapper.tsx`
*   **ツールボタンのスタイル変更**:
    *   現在の「白背景・丸型・影付き (`rounded-full shadow-md`)」から、ヘッダーアイコンに合わせた「四角丸・フラット/ミニマル (`rounded-md`、ボーダー/影の削除または軽量化)」に変更します。
    *   これにより、視覚的な重さを減らし、「縮小」された印象を与えます。
    *   サイズは `p-1` 、アイコンサイズ `w-3.5 h-3.5` (または `w-3 h-3`) を維持しつつ、余計な装飾を削ぎ落とします。

## 検証計画
### 手動検証
1.  **カード編集画面レイアウト**:
    *   カード編集画面を開く。
    *   ヘッダー左上が `[INDEX Q#] [耐性スコア] [縦並びアイコン]` の順になっていることを確認する。
    *   それぞれの高さが視覚的に整っていることを確認する。
    *   その右側にタイトル入力欄があることを確認する。
2.  **ブロックツール表示**:
    *   任意のブロック（テキスト等）にホバーする。
    *   右上に表示されるツール（グリップ、複製、削除）が、以前より小さく（またはヘッダーアイコンと同じサイズ感で）表示されることを確認する。
    *   スタイルが `rounded-md` になり、過度な主張（影など）が抑えられていることを確認する。
