# フォルダ一覧の縦幅を極限まで縮小

## 概要
「もっと縮小して」という要望に応え、視認性をギリギリまで維持しつつ、物理的な高さを最小限にする究極のコンパクト化を行います。

## 変更内容

### Components/folder

#### [MODIFY] [FolderTree.tsx](file:///c:/FlashcardMaster/src/Components/folder/FolderTree.tsx)

- [x] **全体の高さ (Item Container)**:
  - ルートフォルダの下マージン: `mb-1` に縮小。
  - 要素間隔を `gap-1.5` に縮小。

- [x] **アイコンコンテナ (Folder Icon Chip)**:
  - コンテナサイズを `w-6 h-6 md:w-7 md:h-7` に縮小。
  - フォルダアイコンを `w-3 h-3 md:w-3.5 md:h-3.5` に縮小。

- [x] **パディング (Padding)**:
  - ルートフォルダ: `py-0.5 px-1.5 md:py-1 md:px-2`
  - サブフォルダ: `py-0 px-1 md:py-0.5 md:px-1.5` (極限まで最小化)

- [x] **操作ボタン (Action Buttons)**:
  - ボタンサイズを `h-6 w-6` に、アイコンを `w-3.5 h-3.5` に縮小。

- [x] **装飾調整**:
  - 角丸を `rounded-xl` に変更し、タイトなデザインにしました。

## 検証計画
- ローカルサーバーでの表示確認。
- アイコンやテキストの潰れがないか確認。
- 編集モード（Gripやボタンが表示される状態）でのレイアウト崩れがないか確認。
