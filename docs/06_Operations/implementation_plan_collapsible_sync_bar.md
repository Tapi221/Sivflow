# 同期ステータスバーの折りたたみ機能追加

## 目的
同期ステータスバーを折りたたんでアイコンのみのコンパクト表示にできるようにし、画面占有率を下げつつ、必要に応じて詳細を確認できるようにします。

## 提案される変更点

### 同期コンポーネント

#### [MODIFY] [SyncStatusIndicator.tsx](file:///c:/FlashcardMaster/src/Components/sync/SyncStatusIndicator.tsx)
- **状態管理の追加**:
    - `isExpanded` ステータス（デフォルトは `false` または前回の状態を保持）を導入します。
- **UIの変更**:
    - 全体を包むコンテナの外側に、展開/折りたたみを制御する「トリガー」となる要素を配置します。
    - または、ステータスアイコン自体をクリックすることでトグルするようにします。
    - `framer-motion` を使用して、テキスト部分の幅がスムーズに伸縮するようにアニメーションを適用します。
- **自動展開ロジック**:
    - 同期実行中（`syncStatus === 'syncing'`）やエラー発生時は、自動的に展開してユーザーに知らせるオプションを検討します。

### レイアウトの調整

#### [MODIFY] [Layout.tsx](file:///c:/FlashcardMaster/src/Layout.tsx)
- **SyncStatusIndicator の呼び出し修正**:
    - `showText={true}` を `showText={false}` に変更（または削除）し、折りたたみ機能が正しく動作するようにします。
    - `className` に指定されていた背景色や境界線のスタイルを削除し、コンポーネント側のガラスデザインが正しく適用されるようにします。

## 検証計画

### 手動確認
1. バーがアイコンのみの状態で表示されることを確認。
2. クリックするとスムーズに詳細（「最終同期：今日...」など）が展開されることを確認。
3. 再度クリックすると再びアイコンのみに折りたたまれることを確認。
4. 同期ボタンを押した際、同期中は展開状態が維持されるか確認。
