# フォルダ管理画面：フォルダ縦幅縮小

## 概要
フォルダ管理画面における各フォルダの表示高さを縮小し、一覧性を向上させます。

## ユーザーレビューが必要な事項
- `FolderTree.tsx` のスタイル変更は、モバイル・デスクトップ両方に影響します。
- アイコンサイズやパディングの変更が、タッチ操作のしやすさに影響しないか確認が必要です。

## 提案される変更

### Components/folder
#### [MODIFY] [FolderTree.tsx](file:///c:/FlashcardMaster/src/Components/folder/FolderTree.tsx)
- [x] `FolderItem` コンポーネント内のパディング (`p-2.5 md:p-3.5`) を `p-2 md:p-2.5` に縮小。
- [x] マージン (`mb-2`) を `mb-1.5` 程度に縮小。
- [x] アイコンコンテナサイズ (`w-9 h-9 md:w-10 md:h-10`) を `w-8 h-8 md:w-9 md:h-9` に調整。
- [x] フォントサイズや行間 (`gap`) の微調整を行い、バランスを取る。

## 検証計画

### 自動テスト
- 現在、UIの見た目に関する自動テストは存在しないため、手動検証を行います。

### 手動検証
1. `npm run dev` でローカルサーバーを起動。
2. `/folders` (フォルダ管理画面) にアクセス。
3. フォルダの表示高さが縮小されていることを確認。
4. モバイルビュー（幅狭）で表示が崩れていないか確認。
5. フォルダのドラッグ＆ドロップ、展開/折りたたみ、選択などの操作に支障がないか確認。
