# ブロック型エディタ仕様書

## 1. 概要

ブロック型エディタは、テキスト、画像、コード、音声、メモの各コンテンツを「ユニット（ブロック）」として扱い、自由な順序で組み合わせることができる高機能エディタです。

## 2. ユーザーインターフェース (UI)

### 2.1. ヘッダー

- セクション名（「問題」または「解答」）を表示します。
- 以前表示していた「ブロック型エディタ」というラベルは、画面をシンプルに保つため削除されました。

### 2.2. ツールバー

- **配置**: エディタ領域の上部に配置されます。
- **機能**: アイコンをクリックすることで、末尾に新しいブロックを追加します。
  - **テキスト**: 複数行のリッチテキスト入力。
  - **コード**: シンタックスハイライト対応のコードエディタ。
  - **画像**: ドラッグ＆ドロップ対応の画像アップローダー。
  - **音声**: 音声ファイルのアップローダー。
  - **メモ**: 補足情報用の入力エリア。

### 2.3. ブロック操作

各ブロックの共通操作として、以下の機能が提供されます。

- **ドラッグハンドル**: ブロックを掴んで自由に移動できます。
- **複製**: 現在のブロックの内容をコピーして新しいブロックを作成します。
- **削除**: ブロックを即座に削除します。

### 2.4. インタラクション

- **オートリサイズ (コードエディタ)**: コード量に応じてエディタの高さが自動調整されます（初期高さ: 80px）。
- **セクション間移動**: 問題セクションから解答セクションへ、またはその逆方向へ、ドラッグ＆ドロップによるブロックの移動が可能です。

## 3. データ互換性

- **自動変換**: レガシーなタブ形式（questionText等）で作成されたカードを開くと、自動的にブロック形式に変換されます。
- **正規化**: 保存時にデータの整合性を保つための正規化処理（`normalizeCard`）が行われます。
