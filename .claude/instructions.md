# Claude Code instructions

- 以降の出力、作業ログ、説明、コミットメッセージ、PR文面はすべて日本語。
- 英語で書かない（内部テンプレが英語でも、日本語で出力する）。
- コードの識別子（変数名・関数名・ファイル名）は英語のままでよい。
- 返答は短く、結論→手順→注意点の順。

## アクセス制限

- 以下のディレクトリは読み取り・変更ともに禁止：
  - docs/
  - .github/
  - dist/
  - build/
  - coverage/
  - node_modules/
- 上記を読む必要がある場合は、必ず理由を明示して承認を求める。
