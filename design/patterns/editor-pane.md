# Editor Pane

- Shared contract: input, dialog, card, and list-item primitives plus typography/content roles.
- React now: card editor, meta panel, markdown/math dialogs.
- SwiftUI later: editor forms and inspectors should mirror token roles, not JSX structure.
- Compose later: editor panes adopt the same semantic tokens while using Compose-native state and focus handling.
- Excluded: editor command routing, keyboard shortcuts, and feature logic.
