# ChatGPT Approval Hub

複数のブラウザタブで ChatGPT を開いているとき、GitHub などの承認カードを拡張機能のポップアップにまとめて表示し、選択したものを承認するための Chrome 拡張です。

## インストール

1. このリポジトリをローカルに取得します。
2. Chrome で `chrome://extensions` を開きます。
3. 右上の「デベロッパーモード」をオンにします。
4. 「パッケージ化されていない拡張機能を読み込む」を押します。
5. `chatgpt-approval-hub` フォルダを選択します。
6. 既に開いている ChatGPT タブをリロードします。

## 使い方

1. ChatGPT タブで承認カードを表示します。
2. Chrome の拡張機能アイコンから「ChatGPT Approval Hub」を開きます。
3. 一覧に出た承認待ちを確認します。
4. 個別の「承認」または「選択したものを承認」を押します。

## 注意

- ChatGPT の画面構造やボタン文言が変わると検出できなくなる可能性があります。
- 誤承認を避けるため、完全自動承認ではなく、一覧で確認してから押す設計にしています。
- 検出条件を調整したい場合は `content.js` の `APPROVE_WORDS`, `REJECT_WORDS`, `TOOL_HINTS` を編集してください。
